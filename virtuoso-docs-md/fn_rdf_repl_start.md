## Name

RDF_REPL_START — Enables RDF Replication.

</div>

<div>

## Synopsis

<div>

|                             |      |
|-----------------------------|------|
| ` `**`RDF_REPL_START`**` (` | `)`; |

<div>

 

</div>

</div>

</div>

<div>

## Description

This function is called to enable (start) RDF Publication.

</div>

<div>

## Example

<div>

**Example 24.352. Enable RDF Publication**

<div>

``` screen
SQL> DB.DBA.RDF_REPL_START();
```
