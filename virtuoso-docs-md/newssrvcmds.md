### 19.5.2. NNTP Server Commands

</div>

</div>

</div>

The server supports the following commands:

|                               |
|-------------------------------|
| article \[MessageID\|Number\] |
| body \[Number\]               |
| group newsgroup               |
| head \[Number\]               |
| help                          |
| last                          |
| list                          |
| mode reader                   |
| newgroups yymmdd hhmmss       |
| next                          |
| post                          |
| xover \[range\]               |
| stat \[MessageID\|Number\]    |
