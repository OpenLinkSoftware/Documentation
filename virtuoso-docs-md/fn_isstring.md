## Name

isstring — returns true if its argument is of type varchar

</div>

<div>

## Synopsis

<div>

|                              |                      |
|------------------------------|----------------------|
| `boolean `**`isstring`**` (` | `arg ` anything `)`; |

<div>

 

</div>

</div>

</div>

<div>

## Description

isstring returns one if its argument is of type VARCHAR, zero otherwise.

</div>

<div>

## Parameters

<div>

### arg

Some variable to be assessed.

</div>

</div>
