### 19.5.1. Enabling the NNTP Server

</div>

</div>

</div>

The Virtuoso News Server listening port is configured in the HTTP
section of the virtuoso.ini file.

``` screen
[HTTPServer]
NewsServerPort   = 119
```

If the port is not defined Virtuoso will disable the News Server
subsystem.
