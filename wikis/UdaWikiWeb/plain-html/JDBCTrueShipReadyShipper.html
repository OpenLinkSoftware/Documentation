Windows

ReadyShipper is a 32-bit tool for Windows, even when installed in a 64-bit environment.

You will need --

  • the 32-bit Java 8 or later for Windows (direct link to offline installer for Version 8 Update 101). Neither the online installer, nor the 64-bit, will deliver what you need.


  • the 32-bit JDBC Driver for ODBC Data Sources (a/k/a the JDBC-to-ODBC Bridge), Release 7.0 or later, which is provided here with a two-week trial license


  • a 32-bit ODBC driver for your target DBMS or other data source

Then proceed with normal pre-installation and installation, as documented.

Note that ReadyShipper internally handles all interactions with our JDBC-to-ODBC Bridge, including constructing the JDBC URL which is typically constructed manually -- you only need to point ReadyShipper at your ODBC DSN, and provide appropriate username and password.

That said, you may want to test our JDBC-to-ODBC Bridge against your ODBC DSN, without ReadyShipper in the mix, for which you will need to manually construct that URL.
