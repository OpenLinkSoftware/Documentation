Error Message: The installer could not find any packages (.taz) to install.

This message typically indicates that you downloaded the installation archives using Internet Explorer, or some other browser which "helpfully" renamed the archives with different file extensions.

Our installation script only operates on archives with the file extensions as we created them -- i.e., something.taz.

If your download tool changed the names to something.tgz or something.taz.tgz or anything else, all you need do is change the names back to their originals.
