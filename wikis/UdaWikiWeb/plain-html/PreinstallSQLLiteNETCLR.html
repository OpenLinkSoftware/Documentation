OpenLink .NET Data Provider for Microsoft SQL Server Pre-Installation Requirements

Pre-Installation Requirements for .NET CLR Clients

DBMS Requirements

You need to know a variety of information pertaining to your SQL Server DBMS:

  • Microsoft SQL Server instance name
  • Microsoft SQL Server listen port
  • Microoft SQL Server database name
  • Microsoft SQL Server database server hostname or IP address

Software Requirements

  • There are no software requirements associated with the use of OpenLink's .NET Data Provider for Microsoft SQL Server.

Configuration Requirements

  • There are no software configuration requirements associated with the use of OpenLink's .NET Data Provider for Microsoft SQL Server.
