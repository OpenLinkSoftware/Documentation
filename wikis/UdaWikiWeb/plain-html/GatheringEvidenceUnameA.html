Gathering Additional Evidence


uname -a

uname -a is a Linux or Unix command. This command returns information about the operating system, which you are using. To produce a uname, access your Linux or Unix system's command line. Then, execute the following command:

uname -a


Finally, provide the information which it returns. For instance:

Linux nkhatam 2.2.5-16 #1 Tue Apr 27 04:55:03 EDT 1999 alpha unknown
