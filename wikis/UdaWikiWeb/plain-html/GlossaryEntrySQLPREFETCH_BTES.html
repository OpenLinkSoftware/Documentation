SQL_PREFETCH_BYTES

The SQL_PREFETCH_BYTES parameter appears in the [Client] section of your virtuoso.ini file. It passes the maximum number of bytes which your server will send as prefetched rows on a forward only cursor. If long rows are being prefetched SQL_PREFETCH_BYTES will cut off the prefetch after the number of bytes which it specifies.
