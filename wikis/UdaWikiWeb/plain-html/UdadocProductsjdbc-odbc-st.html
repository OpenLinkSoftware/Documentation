These are Type 1 High-Performance Drivers for JDBC that provide transparent access to your ODBC data source via ODBC Drivers. They facilitate access to ODBC Data Sources from JDBC client environments that only support JDBC based data access.

The OpenLink? Single-Tier JDBC-ODBC Bridge Drivers are a Type C class of driver.
