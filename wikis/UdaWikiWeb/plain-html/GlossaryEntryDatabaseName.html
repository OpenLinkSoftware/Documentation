Glossary of Technical Terms

Database Name

The Database Name field or Database parameter appears in a variety of places. You will encounter a Database field or parameter if you try to create DSN's using the ODBC Data Sources Administrator, the Administrative Assistant, the odbc.ini file, the udbc.ini file, or JDBC. You will also find a Database parameter in the generic sections in the OpenLink Session Rules Book and the Administrative Assistant's Database Agent Settings Form and Wizard. You may use the Database Name field or Database parameter to pass the logical name of the database. You may also pass the full path to the database or an Oracle SID.

Be advised. If you hardcode your database name into the Session Rules Book, the Broker will ignore the database name value passed by the client.

Example: Database = /dbs/progress/sports.db

Variants: /DATABASE, Database Name
