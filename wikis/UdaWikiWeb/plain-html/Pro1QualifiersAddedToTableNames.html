Application Specific Problems

pro1 Qualifiers Added To Table Names

Microsoft Access users may discover that Access appends "pro1" qualifiers to their Progress table names. To remedy this problem, users must run the tableview scripts, which OpenLink provides. Moreover, users must configure their Data Sources and Multi-Tier Session Rules Books for tableview connectivity, as appropriate.

See Distributed Databases, Array Fields, Database Triggers for additional information on tableview implementation.
