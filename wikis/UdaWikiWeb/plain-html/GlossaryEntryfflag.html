-f

[1] -f is an option, which may be passed to the OpenLink Request Broker or Virtuoso server at startup. This option causes either service to write debug information to the screen.

Example: ./oplrqb -f

See Also: +foreground

[2] -f is an option which may be passed to oplshut. It causes oplshut to forcibly terminate the OpenLink Broker process. Once employed, the Broker cannot wait for clients to terminate.
