The Multi-Tier JDBC driver for DB2 consists of a set of generic JDBC client classes and the DB2 Database agent on the server utilizing the built in database independent network layer for communication between the client and server.

The OpenLink? Multi-Tier .Net Provider for DB2 are a Type A class of provider developed using the DB2CLI call level interface.

These drivers are currently available for DB2 versions 6.x, 7.x, 8.x and beyond.
