The Multi-Tier .Net Provider for DB2 consists of a 100% managed client assembly and the DB2 Database agent on the server utilizing the built in database independent network layer for communication between the client and server.

The OpenLink? Multi-Tier .Net Provider for DB2 are a Type A class of provider developed using the DB2CLI call level interface.

These drivers are currently available for DB2 versions 6.x, 7.x, 8.x and beyond.


[[/images/aodbcst.gif|]]


/DB2 .Net Provider Architecture Diagram Image scaled down; Click to enlarge./


This driver is available for Windows and Linux Operating systems.
