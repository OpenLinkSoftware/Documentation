Comment

[1] Comment appears as a parameter in the Administrative Assistant DSN creation wizards and the ODBC Data Sources Administrator. In either case, it contains information about the use or purpose of a DSN. It has no bearing on the actual connection.

[2] Comment appears in the Administrative Assistant's Database Agent Settings Form and Wizard. In this instance, it allows you to summarize the use or purpose behind a particular agent configuration stanza. However, Comment has no bearing on the actual connection.
