Installation Shell Scripts

  1. Insure that you have ftp'd your installation shell scripts in ASCII mode and your installers in binary mode. If you ftp'd your installation shell scripts in binary, you may get M^ errors during the install process. This can easily be rectified through repeat ftp in the correct mode.
  2. Obtain copies of your installation shell script and installation files.
  3. Obtain the output of uname -a on the problem machine.
  4. Forward the files to OpenLink Product Support.
  5. Provide the full text of any error that you have encountered.
  6. OpenLink will attempt to run the installation procedure on a similarly spec'd machine.

OpenLink Product Support will raise a bug, if the installation procedure fails on a similarly spec'd machine, and the components are suited for that machine. Support will provide suitable components, if the ones that you provide are inappropriate for your operating system. Support will provide guidance, if you have the correct components, they install inhouse, but they fail in your environment. This is indicative of a configuration or resources problem that can be resolved.
