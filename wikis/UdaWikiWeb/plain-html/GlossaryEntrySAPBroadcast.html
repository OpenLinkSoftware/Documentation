SAPBroadcast

The SAPBroadcast parameter occurs in the [Protocol SPX] section of the OpenLink Session Rules Book. If SAPBroadcast is set to YES, the Request Broker will forward SAP registration packets which contain the name OPENLINK. Local file servers and applications which listen for SAP packets will recognize the packets and become aware of the Request Broker's presence. Finally, the Broker uses the SAPInterval Parameter to determine when packets should be sent.

Example:

SAPBroadcast = Yes
