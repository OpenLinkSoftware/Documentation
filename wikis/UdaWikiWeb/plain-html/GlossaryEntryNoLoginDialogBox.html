No Login Dialog Box

No Login Dialog Box appears on the Configuration tab of Multi-Tier System DSN's. Specifically, it appears on System DSN's which you create using the Microsoft ODBC Data Sources Administrator. If you check this box, login dialogue boxes will not appear, when you use this particular DSN. This is useful, if you hardcode your username and password into your server's OpenLink Session Rules Book.
