Database Agent Administration

Guide to successfully install and run the OpenLink server components from the OpenLink Multi-Tier Data Access Suite. This chapter describes the roles of the OpenLink Database/Service Agents and how to configure them. OpenLink Server Agents come in two main types: the Database agent, and the Service agent. Database agents provide the only database-dependant component, usually requiring a separate agent per major database version. Service agents provide other services such as proxying of connections or special interfaces to other ODBC data sources.
