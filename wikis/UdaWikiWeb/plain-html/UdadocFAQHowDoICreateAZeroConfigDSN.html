doc.FAQ.HowDoICreateAZeroConfigDSN







  • Open your server's Administrative Assistant in a Web Browser.
  • Expand the *Server Components Administration* menu.
  • Expand the Zero Config Administration menu.
  • Click *Zero Config Administration by Wizard* or *Zero Config Administration by Form* .
  • Edit an existing Zero Config template or add a new one.
  • Hard-code the value that your users need to connect to the database.
  • Reinitialize the Request Broker to register your Zero Config DSN.
  • Supply the name of the Zero Config DSN to your end users.
  • Create a new DSN using the Windows Data Source Administrator or the Administrative Assistant.
  • Name the new DSN.
  • Expand the server drop down menu that appears on the first screen of the DSN setup dialog.
  • Locate the Zero Config DSN by name.
  • Allow the Zero Config DSN to auto-fill the fields of the new DSN.
  • Save the new DSN.
  • Test.
