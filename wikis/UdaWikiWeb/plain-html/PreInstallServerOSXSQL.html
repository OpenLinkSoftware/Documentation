Pre-Installation Checklist for Request Broker & Microsoft SQL Server Agents (Mac OS X)

DBMS Requirements

You need to know a variety of information pertaining to your target Microsoft SQL Server DBMS instance and database:

  • Microsoft SQL Server instance name
  • Microsoft SQL Server listen port
  • Microsoft SQL Server database name
  • Microsoft SQL Server database server hostname or IP address

Software Requirements

  • There are no software requirements associated with use of OpenLink's Microsoft SQL Server database agent and Request Broker on Windows.

Configuration Requirements

  • You must know whether TCP ports 5000 and 8000 are in use on the local OS X machine. These ports are used by OpenLink's Request Broker and Web-based Admin Assistant. You will need to allocate different ports for these two services during installation.
