DBMS Requirements

You need to know a variety of information pertaining to your target PostgreSQL instance and database:

  • PostgreSQL TCP port
  • Hostname or IP address of the PostgreSQL database server

Software Requirements

  • You need to know whether the client application is 32-bit or 64-bit. The Single-Tier "Lite" Edition ODBC Driver for PostgreSQL must match the bit format of the client application.

Configuration Requirements

  • There are no software configuration requirements associated with the Single-Tier "Lite" Edition ODBC Driver for PostgreSQL.
