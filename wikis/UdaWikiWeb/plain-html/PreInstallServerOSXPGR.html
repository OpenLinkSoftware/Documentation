Pre-Installation Checklist for Request Broker & PostgreSQL Server Agents (Mac OS X)

DBMS Requirements

You need to know a variety of information pertaining to your target PostgreSQL? DBMS instance and database:

  • PostgreSQL TCP port
  • Hostname or IP address of the PostgreSQL database server

Software Requirements

  • There are no software requirements associated with use of OpenLink's PostgreSQL database agent and Request Broker on Mac OS X.

Configuration Requirements

  • You must know whether TCP ports 5000 and 8000 are in use on the local OS X machine. These ports are used by OpenLink's Request Broker and Web-based Admin Assistant. You will need to allocate different ports for these two services during installation.
