The Multi-Tier .Net Provider for Sybase consists of a 100% managed client assembly and the Sybase Database agent on the server utilizing the built in database independent network layer for communication between the client and server.

The OpenLink? Multi-Tier providers for Sybase are a Type B class driver developed using the TDS network (wire) protocol interface.

These providers are currently available for Sybase versions 4.9, 11, 12 and beyond.
