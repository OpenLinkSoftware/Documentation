Version `SUNW_1.20.4' not found

"Version 'SUNW_1.20.4' not found" suggests that you have a Solaris box that is older than the Solaris box, on which OpenLink's components are built. This is a compatibility problem. To resolve this problem, take the following action:

1. Login to the problem machine.

2. Run the following command: uname -a

3. cd into the /bin sub-directory of the OpenLink installation.

4. Run this command: oplrqb -?

5. Supply the output of uname -a and oplrqb -? to OpenLink Product Support.

Note: In some instances, you may not have an oplrqb executable. In this instance, provide the names of the files that you downloaded from the OpenLink Web site.

For example:

srami9zz.taz

Evidence

uname -a
Name of .taz files
