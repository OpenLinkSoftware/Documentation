ntext data cannot be sent to clients using DB-Library (such as ISQL) or ODBC version 3.7 or earlier

The "ntext" errors arise when OpenLink's Microsoft SQL Server 6 database agents are used to query SQL Server 2000. If you are querying SQL Server 2000, insure that you have the latest OpenLink SQL Server 2000 Database Agent. This Database Agent is specially designed to accommodate ntext and other SQL Server 2000 data types.
