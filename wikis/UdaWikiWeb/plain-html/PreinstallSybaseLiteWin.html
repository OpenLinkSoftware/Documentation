OpenLink .NET Data Provider for Sybase Pre-Installation Requirements

Pre-Installation Requirements for Windows Clients

DBMS Requirements

You need to know a variety of information pertaining to your Sybase DBMS:

  • Sybase instance name
  • Sybase listen port
  • Sybase database name
  • Sybase database server hostname or IP address

Software Requirements

  • You need a functional installation of the .NET frameworks.

Configuration Requirements

  • There are no software configuration requirements associated with the use of OpenLink's .NET Data Provider for Sybase.
