The Single-Tier ODBC Bridge Drivers conforms to the common architectural perceptions of an ODBC driver -- a single component installed on a desktop, workstation, or application server machine.

The OpenLink? Single-Tier ODBC Bridge Drivers are a Type B class of driver bound to the ODBC Driver Manager enabling connectivity to any ODBC enabled Database.
