-119

This error stems from issues in the Release 5.x InstallShield setup.exe which are not seen consistently and thus have not been resolved. Please note
that Release 5.x has generally been deprecated in favor of Release 6.x. Release 6.x does not have this installation issue, because OpenLink shifted from the InstallShield "setup.exe" to the Microsoft Installer.

If you must work with the 5.x installer, you can often work around this issue by performing a Custom Install. When reviewing components for installation simply deselect the OLE DB Client from the components listbox.
