OpenLink Multi-Tier Client Pre-Installation Requirements

Pre-Installation Requirements for Multi-Threaded Generic JDBC Driver (Windows)

DBMS Requirements

  • There are no DBMS requirements associated with the use of OpenLink's Multi-Threaded Generic JDBC Driver.

Software Requirements

  • You need to ensure that you have a functional JVM (JAVA installation).

Configuration Requirements

  • There are no software configuration requirements associated with the use of OpenLink's Multi-Threaded Generic JDBC Driver.
