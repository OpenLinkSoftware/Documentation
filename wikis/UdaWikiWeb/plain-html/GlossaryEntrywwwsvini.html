www_sv.ini

www_sv.ini is a configuration file that contains the HTTP port, username, encrypted password, and other parameters associated with the operation of the HTTP-based iODBC Data Sources Administrator or the Multi-Tier Administrative Assistant. The file resides in the /bin/w3config sub-directory of OpenLink installations. It is associated with the www_sv executable.
