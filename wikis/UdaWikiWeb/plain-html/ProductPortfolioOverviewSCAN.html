OpenLink Product Overview

ODBC Scan

OpenLink Software produces ODBC Scan. ODBC Scan creates clientside ODBC traces that append timestamps to individual ODBC function calls. These timestamps pinpoint API bottlenecks. Users and consultants may also use SCAN to display the following driver information:

* Supported ODBC API function calls and ODBC Level.
* Supported statement options and ODBC Level.
* Supported ODBC data types.
* SQL =>C data type mapping.
