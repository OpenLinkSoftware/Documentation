Can't find oplite_hlp.txt

"Can't find oplite_hlp.txt" is a known error. It occurs when the ODBC Data Sources Administrator searches for a text-based help file. This file previously shipped with OpenLink's products. However, OpenLink stopped shipping this file, once its documentation became available in .html and .pdf format. Finally, OpenLink is implementing code which "grays out" the Data Sources Administrator's "help" button. However, it may not have been incorporated into your driver. Please upgrade or ignore this phenomenon.
