The Multi-Tier ODBC Drivers for CA-Ingres consists of the generic ODBC client driver and the CA-Ingres Database agent on the server utilizing the built in database independent network layer for communication between the client and server.

The OpenLink? Multi-Tier Drivers for CA-Ingres are a Type A class driver developed using the OpenAPI? call level interface.

These drivers are currently available for Ingres II versions 1.x, 2.x and beyond.
