Identifying the Microsoft SQL Server Listen Port

Microsoft SQL Server 2000 and earlier

  1. Log in to your Microsoft SQL Server database server.
  2. Launch the Network Utility (Start menu >> Programs >> Microsoft SQL Server).
  3. Highlight the TCP icon associated with your instance name.
  4. Click Properties.


Microsoft SQL Server 2005 and later

  1. Log in to your Microsoft SQL Server database server.
  2. Launch the Configuration Manager (Start menu >> Programs >> Microsoft SQL Server).
  3. Highlight your Microsoft SQL Server instance name.
  4. Click the TCP/IP icon associated with the instance name.
  5. Click the IP addresses tab on the dialog that appears.
