The Multi-Tier ODBC Drivers for Sybase consists of the generic ODBC client driver and the Sybase Database agent on the server utilizing the built in database independent network layer for communication between the client and server.

The OpenLink? Multi-Tier Drivers for Sybase are a Type B class driver developed using the TDS network (wire) protocol interface.

These drivers are currently available for Sybase versions 4.9, 11, 12 and beyond.
