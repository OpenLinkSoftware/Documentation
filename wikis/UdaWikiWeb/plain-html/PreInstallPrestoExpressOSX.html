DBMS Requirements

You need to know a variety of information pertaining to your PrestoDB DBMS:

  • PrestoDB listen port
  • PrestoDB database/catalog name
  • PrestoDB database server hostname or IP address

As detailed in the PrestoDB documentation.

Software Requirements

  • You must have the 64bit Java SE Development Kit 8uXX for Mac OS X installed for the PrestoDB driver to work, as this is a 64bit only driver.
  • Verify you have the correct Java SE Development Kit 8uXX jdk1.8.0_XX.jdk installed with the command: 


    
$ ls -l /Library/Java/JavaVirtualMachines/
total 0
drwxr-xr-x@ 3 root  wheel  102 26 Nov  2013 jdk1.7.0_45.jdk
drwxr-xr-x  3 root  wheel  102  9 Jan  2015 jdk1.7.0_72.jdk
drwxr-xr-x  3 root  wheel  102  3 Jun 00:45 jdk1.8.0_XX.jdk
$


Configuration Requirements

  • There are no additional software configuration requirements associated with the use of the "Express" Edition ODBC Driver for PrestoDB.
