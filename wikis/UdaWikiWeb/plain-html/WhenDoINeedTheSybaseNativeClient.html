When do I need the Sybase native client?

OpenLink's current data access products do not use the Sybase native client. Use of the Sybase native client was deprecated with the advent of OpenLink 4.0.

There are two scenarios in which you would require the Sybase native client. You must install the Sybase client on the same machine as any 1.x or 3.x Single-Tier driver. Additionally, you must install the Sybase client on the same machine as OpenLink's Request Broker and Sybase Database Agent, if the Broker and Sybase Agent do not reside on the same machine as the Sybase database, and the Sybase Agent is version 1.x or 3.x.
