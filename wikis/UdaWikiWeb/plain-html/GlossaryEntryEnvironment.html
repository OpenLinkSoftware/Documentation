Environment

[1] The [Environment] sections appear in the OpenLink Session Rules Book. They contain the environmental variables, which must be set in order for an agent to connect to a particular database or to connect in a particular environment. For instance, Oracle connections require the ORACLE_HOME and ORACLE_SID variables. Progress connections require DLC.

[2] An Environment parameter appears in each [generic_xxx] section in the OpenLink Session Rules Book. This parameter passes the name of an [Environment] section, which contains variables that must be set for an agent to function. The Environment parameter corresponds to the Environment Variables list boxes, which you may encounter in the Administrative Assistant.

Example: DLC = C:\DLC
