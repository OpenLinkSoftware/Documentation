DBMS Requirements

You need to know a variety of information pertaining to your Ingres node and database:

  • The Ingres node name
  • The Ingres listen address
  • The Ingres database name

Software Requirements

  • You must have the Ingres native client installed on the same machine as OpenLink's Ingres Lite drivers. You will find your Ingres II native client on your Ingres install media. The Ingres 3 client is featured in the Open Source download kit. At the time of writing, pre-compiled library binaries were missing for many platforms on which Ingres 3 runs.

Configuration Requirements

  • You need to configure a local Ingres vnode.
