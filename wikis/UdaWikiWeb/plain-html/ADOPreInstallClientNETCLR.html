OpenLink Generic ADO.NET Data Provider Pre-Installation Requirements

Pre-Installation Requirements for .NET CLR Clients

DBMS Requirements

  • There are no DBMS requirements associated with the use of OpenLink's Generic ADO.NET Data Provider.

Software Requirements

  • You need a functional .NET CLR runtime.
  • You need to know whether the client application is 32 or 64bit. OpenLink's provider must match the bit format of the client application.

Configuration Requirements

  • There are no software configuration requirements associated with the use of OpenLink's Generic ADO.NET Data Provider.
