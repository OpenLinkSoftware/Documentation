DBMS Requirements

You need to know a variety of information pertaining to your Sybase DBMS:

  • Sybase instance name
  • Sybase listen port
  • Sybase database name
  • Sybase database server hostname or IP address

Software Requirements

  • There are no software requirements associated with the Single-Tier "Lite" Edition ODBC Driver for Sybase.

Configuration Requirements

  • There are no software configuration requirements associated with the Single-Tier "Lite" Edition ODBC Driver for Sybase.
