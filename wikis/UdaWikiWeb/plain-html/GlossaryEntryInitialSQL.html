Initial SQL

The Initial SQL field appears in the Administrative Assistant's Database Agent Settings Form and Wizard. You may use this field to pass the full path to a file, which contains SQL statements. Each time a connection is established with the corresponding agent, these SQL statements will be executed. This feature is commonly used to set isolation levels.

See Also: -i, +initsql
