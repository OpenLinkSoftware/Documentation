When do I need the Progress native client?

  • When using any Single-Tier data access driver for Progress, the Progress native client must be installed on the same machine as the Single-Tier driver.
  • When using any Multi-Tier data access driver for SQL-89 connections to Progress, and the OpenLink Request Broker and Progress-specific Database Agent are not installed on the Progress server host, the Progress native client (Progress® Client Networking) must be installed on the same machine as the Broker and Agent.
  • When using any Multi-Tier data access driver for SQL-92 connections to Progress or OpenEdge, the Progress native client (Progress® Client Networking) must be installed on the same machine as the OpenLink Request Broker and Progress-specific Database Agent, including when this is also the Progress server host.
