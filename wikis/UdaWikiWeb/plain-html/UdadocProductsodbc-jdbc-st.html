The Single-Tier ODBC-JDBC Bridge Drivers conforms to the common architectural perceptions of an ODBC driver -- a single component installed on a desktop, workstation, or application server machine.

The OpenLink? Single-Tier ODBC-JDBC Bridge Drivers are a Type B class of driver bound to the JDBC Driver Manager enabling connectivity to any JDBC enabled Database via ODBC.
