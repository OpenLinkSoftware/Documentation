DBMS Requirements

You need to know a variety of information pertaining to your target Sybase instance and database:

  • Sybase instance name
  • Sybase listen port
  • Sybase database name
  • Sybase database server hostname or IP address

Software Requirements

  • You need to know whether the client application is 32-bit or 64-bit. The Single-Tier "Lite" Edition ODBC Driver for Sybase must match the bit format of the client application.

Configuration Requirements

  • There are no software configuration requirements associated with the Single-Tier "Lite" Edition ODBC Driver for Sybase.
