IPAddress

The IPAddress parameter occurs in the [Protocol TCP] section of the OpenLink Session Rules Book. If you have multiple network interface cards in your server, you must set IPAddress to the value of one card. Then, you must pass the same IP address in your clients' DSNs. If you do not set IPAddress, your Request Broker may bind to the wrong interface, and clients will not be able to locate it.

Example: IPAddress = 38.242.233.173
