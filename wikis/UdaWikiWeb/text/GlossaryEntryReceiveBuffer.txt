==Receive Buffer==

The Receive Buffer parameter appears in the Administrative Assistant's Edit Request Broker's Parameters Form and Wizard. Receive Buffer represents the size of the server's internal, RPC receive buffer. (This buffer is used to store incoming information from clients.) Furthermore, your client Receive Size and your server's Receive Buffer values should be identical. However, you do not want to alter these parameters without guidance from <nop>OpenLink Product Support 

Variants: Receive Size 

See Also: Receive Size, Send Size
