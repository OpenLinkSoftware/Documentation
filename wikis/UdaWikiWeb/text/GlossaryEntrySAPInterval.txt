==SAP Interval==

The SAP Interval parameter (<nop>SAPInterval) appears in the [Protocol SPX] section of the <nop>OpenLink Session Rules Book. If <nop>SAPBroadcast is set to Yes, SAP registration packets will be forwarded at intervals specified by SAP Interval. The integer value passed to SAP Interval represents seconds. 

Example: 

{{{
SAPInterval = 100 
}}}

See Also: <nop>SAPBroadcast, SAP Interval, SAP Queries, SAP Server Type, SAP Service Name, SPX
