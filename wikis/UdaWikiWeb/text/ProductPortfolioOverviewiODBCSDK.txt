==<nop>OpenLink Product Overview==

===The <nop>iODBC SDK===

The <nop>iODBC SDK contains a runtime <nop>iODBC Driver Manager, ODBC client configuration files, and the odbctest application. The SDK also contains header files, driver manager and odbctest source code, and LGPL and BSD licensing agreements. Developers can use the headers and source code to create their own ODBC compliant applications.
