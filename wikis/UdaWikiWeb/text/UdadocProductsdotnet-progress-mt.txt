The Multi-Tier .Net Provider for Progress consists of a 100% managed client assembly and the Progress Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for Progress are a Type A class driver developed using the ESQL/C and SQL-92 call level interface. 

These drivers are currently available for Progress versions 7.x, 8.x, 9.x and beyond.
