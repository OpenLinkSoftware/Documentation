==INFORMIXSERVER==

The INFORMIXSERVER environment variable passes the name of the Informix server. This is not the physical machine that hosts Informix. This is the name of the Informix DBMS server, which appears in the Informix Server Instances utility (Windows), Informix user .profile (Unix), and $INFORMIXDIR/etc/sqlhosts file.
