==Trace File==

The Trace File (<nop>TraceFile) parameter is located in the <nop>OpenLink odbc.ini file. If tracing is enabled on Unix systems, ODBC calls will be written into the file specified by the Trace File parameter. You may alter the path and file name to suit your own needs.   

Example: 

{{{
TraceFile = /tmp/iodbc.trace 
}}}

See Also: Debug File, Trace
