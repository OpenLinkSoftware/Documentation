The Multi-Tier ODBC Bridge Driver consists of the generic ODBC client driver and the ODBC Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier ODBC Bridge Drivers are a Type C class driver bound to the ODBC Driver manager enabling connectivity to third party ODBC DSNs. 

 
 [[/images/aodbcst.gif|]]  

 
/MS SQLServer Multi-Tier Architecture Diagram Image scaled down; Click to enlarge./ 

 
This driver is available for Windows, Linux, Unix(most flavours) and Mac OS X Operating systems.
