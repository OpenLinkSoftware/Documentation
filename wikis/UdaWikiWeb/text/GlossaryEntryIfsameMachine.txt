==Ifsame machine==

Ifsame machine is a database agent <nop>ReUse rule value. Ifsame machine specifies that all requests from the same machine will be serviced by one agent. Be advised, machine aliases have no bearing on this rule. Instead, network addresses are translated into machine names, which the Broker will compare. Moreover, if the address cannot be resolved, the Broker will label the machine 'unknown.'
