==Glossary of Technical Terms==

==Debug File==

A Debug File (<nop>DebugFile) parameter appears on the ODBC tabs of the <nop>OpenLink Client Configurator and the Administrative Assistant's Edit Data Sources Wizard utility. A <nop>DebugFile parameter also occurs in the [UDBC] section of udbc.ini. If you pass the full path to a file to Debug File, ODBC trace output will appear in the file. Leave Debug File blank, if you do not want to incur tracing overhead.
