You need to ensure that your 3rd party drivers connect to the target database. 

You need to ensure that openlink.sh or openlink.csh is run on Unix machines that contain [[OpenLink]]  Software. openlink.sh should be run in a Bourne or Bash shell. openlink.csh should be run in C shell.
