= Administration of the Multi-Tier Request Broker through Admin Assistant Forms =

You can configure your Request Broker via Forms [[UsingAdminAssistantWizards| or Wizards]]. You select this option from the Admin Assistant and you will be presented with a screen similar to the snapshot below.

%BR%%BR%<img src="%ATTACHURLPATH%/AAForm1.png" style="wikiautogen"/>%BR%%BR%

Each Broker session settings field contains a description of the values that need to be entered into each of the forms fields. In addition a brief description of the use of each of these values is provided below each field.
