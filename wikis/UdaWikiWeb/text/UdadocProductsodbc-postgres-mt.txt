The Multi-Tier ODBC Drivers for Postgres consists of the generic ODBC client driver and the Postgres Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for Postgres are a Type B class driver developed using the Postgres network (wire) protocol interface. 

These drivers are currently available for Postgres versions 6.x, 7.x and beyond. 

 
 [[/images/aodbcst.gif|]]

This driver is available for Windows, Linux, Unix(most flavours) and Mac OS X Operating systems.
