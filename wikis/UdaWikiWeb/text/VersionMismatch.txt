===Version Mismatch/Shared Memory Mismatch===

If you receive "version mismatch" or "shared memory mismatch" error messages, you will need to [[RelinkMulti-TierDatabaseAgents|relink your database agent]] or switch to [[ServerAgentAdminProgress|sockets-based connections]].

===Evidence===
     
* Database Agent Version String 
* DBMS Version
* Request Broker Log
