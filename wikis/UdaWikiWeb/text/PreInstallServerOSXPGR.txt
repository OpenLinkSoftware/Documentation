===Multi-Tier Pre-Installation & Configuration Docs===

====Pre-Installation Checklist for Request Broker & <nop>PostgreSQL Server Agents (Mac OS X)====

==== DBMS Requirements ====

You need to know a variety of information pertaining to your target <nop> PostgreSQL DBMS instance and database:

* [[HowDoIFindMyPostgreSQLListeningPort|<nop>PostgreSQL TCP port]]
* Hostname or IP address of the <nop>PostgreSQL database server

==== Software Requirements ====

*  There are no software requirements associated with use of <nop>OpenLink's <nop>PostgreSQL database agent and Request Broker on Mac OS X.

==== Configuration Requirements ====

* You must know whether TCP ports 5000 and 8000 are in use on the local OS X machine. These ports are used by <nop>OpenLink's Request Broker and Web-based Admin Assistant. You will need to allocate different ports for these two services during installation.
