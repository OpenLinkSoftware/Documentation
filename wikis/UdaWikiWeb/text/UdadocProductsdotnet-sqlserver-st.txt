The Single Tier .Net Provider for MS SQLServer conforms to the common architectural perception of a .Net Provider -- a 100% Managed Provider installed on a desktop, workstation, or application server machine. 

The [[OpenLink]]  Single-Tier .Net Provider for MS SQLServer are a Type B class of driver developed using the MS SQLServer TDS network (wire) protocol interface eliminating the need for the installation of any MS SQLServer specific client networking software. 

The drivers are currently available for MS SQLServer versions 6.5, 7, 2000 and beyond.
