==Ifsame opsys==

Ifsame opsys is a database agent <nop>ReUse rule value. Ifsame opsys specifies that all requests from the same operating system class (e.g., msdos, win32, unix) will be serviced by one database agent. Be advised, opsys aliases do not apply.
