==Installation Directory==

The Installation Directory field appears in the Administrative Assistant's Edit Request Broker's Parameters Form. This parameter must be set to the /bin sub-directory of your <nop>OpenLink installation. The Broker uses the location passed by this parameter to locate the database agents and Session Rules Book. The Broker will also create log files in this directory, by default.
