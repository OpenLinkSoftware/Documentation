=Unable to start the requested database agent=

"Unable to start" messages are common and generic. They do not reveal the true error condition. These errors are typically preceded and/or followed by more informative error messages.

Common underlying issues involve <nop>ReUse and <nop>ReceiveTimeout settings in the Broker rulebook.
    

===Evidence===

* Request Broker Log 
* Request Broker Version String 
* License File
