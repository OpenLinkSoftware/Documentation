==UDBCINI==

[1] UDBCINI is a Unix environmental variable. It passes the full path to the udbc.ini file which normally resides in the /bin directory of your <nop>OpenLink installation. UDBCINI may be manually set or exported from the command line. It may also be set from users' .profiles. Finally, both openlink.sh and openlink.csh will set the variable. This is the preferred method.   

[2] A UDBCINI environmental variable exists in the [Environment JDBC] section of the <nop>OpenLink Session Rules Book. This variable is also set to the absolute path to udbc.ini. 

See also openlink.csh, openlink.sh, udbc.ini
