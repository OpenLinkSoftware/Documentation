==<nop>PortHigh==

The <nop>PortHigh parameter occurs in the [Protocol TCP] section of the <nop>OpenLink Session Rules Book. The value passed to this parameter represents the highest TCP port which the Request Broker may open for the client and agent to transmit data. 

Example: 

{{{
PortHigh = 5025 
}}}
