===/OPTIONS===

/OPTIONS is a Type 3, JDBC connect string parameter. You may use /OPTIONS to pass any valid, database specific connection parameters. For instance, you may pass Progress sockets options. Alternatively, you may pass -S server_name to designate a particular SQLServer service.   

Example: {{{/OPTIONS = -S pro8 -N tcp -H openlink }}}  

See Also: JDBC, Type 3
