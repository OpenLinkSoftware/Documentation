==INFO==

INFO is a Request Broker log detail setting. It appears in the Administrative Assistant's Edit Request Broker's Parameters Form and Wizard. If you set your Broker's log detail to INFO, the Broker will write informational, warning, and error-related messages to its log file. 

This setting is not optimal for diagnostic purposes.
