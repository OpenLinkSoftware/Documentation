The Multi-Tier ODBC Drivers for Informix consists of the generic ODBC client driver and the Informix Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for Informix are a Type A class driver developed using the ESQL/C call level interface. 

These drivers are currently available for Informix versions 7.x, 9.x(IDS 2000) and beyond. 

 
 [[/images/aodbcst.gif|]]  

 
/Informix Multi-Tier Architecture Diagram Image scaled down; Click to enlarge./ 

 
This driver is available for Windows, Linux, Unix(most flavours) and Mac OS X Operating systems.
