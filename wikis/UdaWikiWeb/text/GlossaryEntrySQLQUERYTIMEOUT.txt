==SQL_QUERY_TIMEOUT==

The SQL_QUERY_TIMEOUT parameter appears in the [Client] section of your virtuoso.ini file. The ODBC standard SQL_QUERY_TIMEOUT value is 0 or "indefinite." However, this setting is impractical in many cases. Therefore, Virtuoso's SQL_QUERY_TIMEOUT allows you to set the initial timeout value--in seconds--for connected clients. Be advised, if a client application passes its own SQL_QUERY_TIMEOUT value, it will override the value passed in the virtuoso.ini file.
