===Cannot link <nop>OpenServer object into Intel iABI target===

"Cannot link <nop>OpenServer object into Intel iABI target" errors are serious. They occur when your Progress database and your <nop>OpenLink Progress Database Agent are linked against different Application Binary Interfaces. Contact <nop>OpenLink Product Support, to determine if new agents have been built to accommodate this version of Progress.
    
===Evidence===

uname -a
Database Agent Version String
DBMS Version
