The Multi-Tier JDBC driver for [[MySQL]]  consists of a set of generic JDBC client classes and the [[MySQL]]  Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for [[MySQL]]  are a Type B class driver developed using the [[MySQL]]  network (wire) protocol interface. 

The drivers are currently available for [[MySQL]]  versions 3.x, 4.x and beyond. 

 
 [[/images/aodbcst.gif|]]  

 
/MySQL Multi-Tier JDBC Driver Architecture Diagram Image scaled down; Click to enlarge./ 

 
This driver is available for Windows, Linux, Unix(most flavours), Mac Classic, Mac OS X or any Java enabled Operating systems.
