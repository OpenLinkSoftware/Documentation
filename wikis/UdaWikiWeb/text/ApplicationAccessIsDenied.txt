===Application access is denied===

"Application access is denied" errors occur, when your <nop>OpenLink license restricts the applications that you can use to query your database via ODBC. If you receive this problem, contact Account Management to ascertain what applications are permitted by your license.

If the problem persists, contact <nop>OpenLink Product Support. You may have a corrupt license, an inappropriate license, a flaw in a Multi-Tier Session Rule, or a Broker that is not properly evaluating the license.
    
===Evidence===

* License File
* Request Broker Log
* Request Broker Version String 
* Request Broker Session Rulebook
* Output of {{{uname -a}}}
