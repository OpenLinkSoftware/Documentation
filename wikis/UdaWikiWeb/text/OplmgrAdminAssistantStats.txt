==<nop>OpenLink License Manager Usage Notes==
===<nop>OpenLink HTTP-based Admin Assistant License Statistics===

The <nop>OpenLink HTTP based Admin Assistant shipped with the Multi-Tier product provides a License statistics page to enable the details of the licenses in use on a given machine to be determined. To view this page start the <nop>OpenLink Request Broker running, load the <nop>OpenLink HTTP based Admin Assistant (http://hostname:portno ; note portno is 8000 by default) and click on the <b><code>Server Components Administration -> General Request Broker Information -> View Server's License Information</code></b> links in the left frame: %BR%
%BR%
<img src="%ATTACHURLPATH%/adminlicdetails.png" style="wikiautogen"/>

Clicking on the "details" link enables more detailed information to be obtained specified license: %BR%
%BR%
<img src="%ATTACHURLPATH%/adminlicstats.png" style="wikiautogen"/>

Note these pages display the number of core processors (CPUs) the License Manager has detected on the host it is running on.
