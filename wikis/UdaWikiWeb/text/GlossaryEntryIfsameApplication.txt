==Ifsame application==

Ifsame application is a database agent <nop>ReUse rule value. Ifsame application specifies that all requests with the same application identifier will be serviced by the same agent. Be advised, application aliases have no bearing on this rule. Instead, the Broker polls for the physical task name used to identify the client application.
