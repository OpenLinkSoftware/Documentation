**DB2 users must pass the following value:*  
*A remote DB2 database alias  
**Informix users must pass the following values:*  
*The Informix server name  
*The Informix server physical hostname or IP address  
*The Informix database name  
*The Informix server protocol  
*The Informix server socket service  
*The Informix Client Local  
**Ingres users must pass the following value:*  
*A vnode::database pair  
**MS SQLServer users must pass the following values:*  
*An [[OpenLink]]  " [[ServerType]]  "  
*-S followed by the SQLServer instance name  
*The SQLServer listening port  
** [[MySQL]]  users must pass the following values:*  
*The [[MySQL]]  host's IP address or hostname  
*The [[MySQL]]  listening port  
*The [[MySQL]]  database  
**Oracle users must pass the following value:*  
*A Net8 or SQL*Net alias  
** [[PostgreSQL]]  users must pass the following values:*  
*The [[PostgreSQL]]  host's IP address or hostname  
*The [[PostgreSQL]]  listening port  
*The [[PostgreSQL]]  database  
**Progress 4GL users must pass the following values:*  
*-S socket -N tcp -H hostname  
**Progress SQL92 & Mixed Mode users must pass the following values:*  
*The Progress database name (full path on Unix)  
*The Progress database server hostname or IP address  
*The Progress socket service  
**Sybase users must pass the following values:*  
*An [[OpenLink]]  " [[ServerType]]  "  
*A Sybase instance name  
*The Sybase listening port
