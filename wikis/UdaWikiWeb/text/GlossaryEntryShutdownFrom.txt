==<nop>ShutdownFrom==

The <nop>ShutdownFrom parameter occurs in the [Security] section of the <nop>OpenLink Session Rules Book. You may set this parameter to a comma-delimited list of machine names, IP addresses, or node names from which the Request Broker may be shut down.   

Example: 

{{{
ShutdownFrom = localhost   
}}}

See Also: <nop>ShutdownBy, <nop>StartupBy
