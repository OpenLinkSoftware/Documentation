==<nop>OpenLink Product Overview==

===The <nop>iODBC Driver Manager===

The <nop>iODBC Driver Manager is a platform-independent, open source implementation of Microsoft's ODBC Driver Manager. A runtime <nop>iODBC Driver Manager is bundled with each of <nop>OpenLink's non-Windows client drivers and iODBC SDKs. Driver Manager source code is also contained within <nop>iODBC SDKs.
