==www_sv.log==

www_sv.log is a log file that contains operational information and errors associated with the operation of the HTTP-based iODBC Data Sources Administrator or the Multi-Tier Administrative Assistant. The file resides in the /bin/w3config sub-directory of <nop>OpenLink installations. It is associated with the www_sv executable www_sv.ini configuration file.
