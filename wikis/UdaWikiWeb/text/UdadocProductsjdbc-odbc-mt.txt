The Multi-Tier JDBC driver for ODBC consists of a set of generic JDBC client classes and the ODBC Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier JDBC-ODBC Bridge Drivers are a Type C class driver bound to the ODBC Driver manager enabling connectivity to third party ODBC DSNs. 

 
 [[/images/aodbcst.gif|]]  

 
/ODBC Multi-Tier JDBC Driver Architecture Diagram Image scaled down; Click to enlarge./ 

 
This driver is available for Windows, Linux, Unix(most flavours), Mac Classic, Mac OS X or any Java enabled Operating systems.
