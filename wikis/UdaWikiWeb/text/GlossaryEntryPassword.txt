==Password==

Password parameters and fields are a widespread occurrence. Typically, passwords are authenticated by your backend database. However, systems which employ OPSYSLOGIN necessitate that you pass an operating system username and password combination. 

See Also: Operating System Identity, OPSYSLOGIN
