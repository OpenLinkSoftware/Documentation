==Executable Name==

The Executable Name field occurs in both the Administrative Assistant's Database Agent Settings Form and Wizard. It corresponds to the Program parameter, which appears in the [generic_xx] sections of your <nop>OpenLink Session Rules Book. This field specifies which agent executable is to be spawned for a given [generic_xxx] section.
