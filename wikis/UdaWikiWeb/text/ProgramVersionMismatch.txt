===Program/version mismatch===

"Program/version mismatch" errors arise when <nop>OpenLink server components are incompatible with one another. The error may occur in the context of a connection, or it may occur when you attempt to run oplshut. In either case, it is best to download the latest client and server components at the same time. This insures that you have fully compatible software. If you need to preserve legacy, custom, or patched software, contact <nop>OpenLink Product Support.
