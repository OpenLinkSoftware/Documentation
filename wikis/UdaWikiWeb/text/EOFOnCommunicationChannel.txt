===End-of-file on communication channel===

"End-of-file on communication channel" errors occur when <nop>OpenLink's Oracle database agents are incompatible with an Oracle database. If you receive "End-of-file" errors, relink your <nop>OpenLink Database Agent against your Oracle database. Furthermore, insure that you have <nop>OpenLink's 4.0 software, if you intend to connect to an Oracle 8i database.
    
===Evidence===

Session Rules Book
Database Agent Version String 
DBMS Version
