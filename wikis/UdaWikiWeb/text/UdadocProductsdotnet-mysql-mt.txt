The Multi-Tier .Net Provider for [[MySQL]]  consists of a 100% managed client assembly and the [[MySQL]]  Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for [[MySQL]]  are a Type B class driver developed using the [[MySQL]]  network (wire) protocol interface. 

The drivers are currently available for [[MySQL]]  versions 3.x, 4.x and beyond.
