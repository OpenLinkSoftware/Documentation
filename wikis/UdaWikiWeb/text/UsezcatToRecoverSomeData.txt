===Corrupt input. Use zcat to recover some data.===

"Corrupt Input" errors occur when <nop>OpenLink binaries are ftp'd from one machine to another, in ASCII format. Please ftp all files in binary mode.

If ftp is not at fault, insure that the <nop>OpenLink components are compatible with the operating system on which you are trying to install them.

    
===Evidence===

uname -a
Name of .taz files
