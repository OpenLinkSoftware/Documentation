The User Name field or <nop>UserName parameter appears in the Administrative Assistant, odbc.ini file, udbc.ini file, and ODBC Data Sources Administrator.  The User Name is used for authentication purposes. If OPSYSLOGIN is enabled, the operating system will authenticate the name. If OPSYSLOGIN is disabled, the name will be authenticated by the backend database.   

See Also: UID, USER_ID 

Variants: <nop>UserName, Username
