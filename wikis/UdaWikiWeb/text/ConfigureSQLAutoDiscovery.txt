==Configure Microsoft SQL Server Auto Discovery ==


Use the following steps to configure Microsoft SQL Server Auto Discovery.

===Single-Tier Drivers===

1. Log in to the machine that hosts <nop>OpenLink's Single-Tier (Lite Drivers).

2. Launch your ODBC or iODBC Data Sources Administrator.

3. Locate your <nop>OpenLink DSN.

4. Click the Configure button.

5. Scroll through the configuration screens until you find the Server Name drop-down menu.

6. Use the drop-down menu to  invoke the driver's Dynamic discovery feature.

7. Select from the available Microsoft SQL Server instances.
