==UID==

The UID parameter appears in UDBC and JDBC connection URL's. If OPSYSLOGIN is enabled, the operating system will authenticate the value passed by UID. If OPSYSLOGIN is disabled, the UID will be authenticated by the backend database.   

See Also: User Name, User_ID
