==Read-Only==

Read-Only is a wide spread parameter with many variants. If Read-Only is enabled, connections to the backend database will prohibit updates to that database. In other words, select statements are the only statements which you may run. Be advised, if you set the Read-Only parameter to Yes, within your Session Rules Book, all connections will be forced to Read-Only mode.
