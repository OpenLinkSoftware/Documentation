==Operating System Identity==

An Operating System Identity checkbox appears in the Administrative Assistant's Database Agent Settings Wizard and Form. If this box is checked, the Request Broker will test to see that the username and password which are supplied to it are valid for the machine. If the authentication parameters are valid, it will start the database agent under the identity of the user. If the Operating System Identity box is not checked, the username and password will be passed to the backend database for authentication.
