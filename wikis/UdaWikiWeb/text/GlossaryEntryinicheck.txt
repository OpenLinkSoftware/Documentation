==inicheck==

The inicheck utility ships with <nop>OpenLink's Unix server components. This utility reads the information contained in your Rules Book and displays the portions of the Rules Book, which are not syntactically correct. (Only logic is examined. Actual values are not.) Finally, inicheck should appear in the /bin sub-directory of your <nop>OpenLink server components installation. 

See Also: Rules Book Checker
