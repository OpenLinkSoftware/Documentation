===-461===


If your Informix 7.3 database produces -461 errors, relink your <nop>OpenLink Database Agent.

Be advised. Some versions of Informix 7.3 utilize 7.2 ESQL/C libraries. If your Informix installation contains 7.2 libraries, download an <nop>OpenLink 7.2 database agent. Then, relink the Agent.

    
===Evidence===

* Request Broker Log 
* Database Agent Version String 
* DBMS Version
