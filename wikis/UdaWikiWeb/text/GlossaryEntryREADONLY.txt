==/READONLY==

/READONLY is a Type 3, JDBC connect string parameter. If you pass /READONLY{{{=}}}YES, connections to the backend database will prohibit updates to that database. In other words, select statements are the only statements, which you may run. If you pass /READONLY{{{=}}}NO, you will have READ/WRITE access to that database, unless serverside settings exclude this. 

Example: /READONLY {{{=}}} NO 

See Also: JDBC, Type 3
