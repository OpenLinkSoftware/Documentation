=TDSVER=

The <code>TDSVER</code> environment variable is used to set the TDS Protocol version to use when connecting to your desired target Sybase or Microsoft SQL Server instance, when the <code>-V</code> value is not set in the client DSN <code>Options</code> or Rulebook <code>Connect<nop>Options</code>.

See [[TDSVersionsExplained|TDS Versions Explained]] for details.
