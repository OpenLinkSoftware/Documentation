==Never==

Never is a database agent <nop>ReUse rule value. Never specifies that a new database agent will be spawned for each client connection. Database agents will not be reused. Be advised, you must insure that you have allocated appropriate resources, if you want to employ the Never <nop>ReUse rule in a high usage environment.
