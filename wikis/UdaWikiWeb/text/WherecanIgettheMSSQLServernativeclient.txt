==Obtain the Microsoft SQL Server native client (a.k.a "<nop>NetLib") ==

<nop>OpenLink Software does not supply the Microsoft SQL Server native client. Instead, the Microsoft SQL Server native client should ship with the standard Microsoft SQL Server DBMS install media.

If you do not have the Microsoft SQL Server client install CD, you should contact Microsoft.
