===Agent Needs Newer Broker===

If you receive "Agent needs newer broker" error messages, your Request Broker and Database Agent versions are incompatible. Forward your Request Broker and Database Agent version strings to <nop>OpenLink Product Support.

===Evidence===

* Request Broker Version String (All Platforms)
* Database Agent Version String (All Platforms)
