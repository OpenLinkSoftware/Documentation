==ORACLE_HOME==

The ORACLE_HOME variable is set in the [Environment ORACLEX] section of the <nop>OpenLink Session Rules Book. This variable is set to the home directory of your Oracle installation. Moreover, it should match the ORACLE_HOME parameter passed in your Oracle user's .profile. 

Note: This variable may be set using the Administrative Assistant's Database Agent Settings Form or Wizard.
