==Ifsame process==

Ifsame process is a database agent <nop>ReUse rule value. Ifsame process specifies that all requests from the same process ID will be serviced by one database agent. This value is similar in concept to ifsame application. 

See Also: Ifsame application, <nop>ReUse Rule
