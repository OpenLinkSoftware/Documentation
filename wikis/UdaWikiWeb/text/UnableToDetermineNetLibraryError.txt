===Unable to determine Net-Library error==

Use the following steps to eradicate this error:

1. Ensure that your Sybase user's ~\SYBASE\DLL directory appears in the system and user paths.

2. Ensure that you are passing a valid Sybase instance name (DSQUERY) where prompted.
