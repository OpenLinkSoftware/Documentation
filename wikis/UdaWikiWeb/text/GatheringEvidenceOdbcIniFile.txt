==Gathering Additional Evidence==
===odbc.ini File ===

The <nop>OpenLink odbc.ini file is a Linux or Unix based configuration file. It appears in the /bin sub-directory of Multi-Tier and Single-Tier installations. Copy this file and forward it to <nop>OpenLink Product Support.
