= Identifying the Microsoft SQL Server Listen Port =

== Microsoft SQL Server 2000 and earlier ==

# Log in to your Microsoft SQL Server database server.
# Launch the **Network Utility** (**Start menu** >> **Programs** >> **Microsoft SQL Server**).
# Highlight the **TCP** icon associated with your **instance name**.
# Click **Properties**.


== Microsoft SQL Server 2005 and later ==

# Log in to your Microsoft SQL Server database server.
# Launch the **Configuration Manager** (**Start menu** >> **Programs** >> **Microsoft SQL Server**).
# Highlight your Microsoft SQL Server **instance name**.
# Click the **TCP/IP** icon associated with the instance name.
# Click the **IP addresses** tab on the dialog that appears.
