= Pre-Installation Requirements for the Single-Tier "Lite" Edition ODBC Driver for Microsoft SQL Server, for Windows =

%TOC%

== DBMS Requirements==

You need to know a variety of information pertaining to your target Microsoft SQL Server instance and database:

* The [[HowCanIDetectMyMSSQLServerDBMSVersion|Microsoft SQL Server version]]
* [[HowDoIFindMySQLServerInstanceName|Microsoft SQL Server instance name]]
* [[HowDoIFindMySQLServerListenPort|Microsoft SQL Server listen port]]
* Microsoft SQL Server database name
* Microsoft SQL Server database server hostname or IP address

== Software Requirements ==

* You need to know whether the client application is 32-bit or 64-bit. The Single-Tier "Lite" Edition ODBC Driver for Microsoft SQL Server must match the bit format of the client application.

== Configuration Requirements ==

* There are no software configuration requirements associated with the Single-Tier "Lite" Edition ODBC Driver for Microsoft SQL Server.
