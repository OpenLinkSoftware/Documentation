The Multi-Tier ODBC Drivers for [[MySQL]]  consists of the generic ODBC client driver and the [[MySQL]]  Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for [[MySQL]]  are a Type B class driver developed using the [[MySQL]]  network (wire) protocol interface. 

The drivers are currently available for [[MySQL]]  versions 3.x, 4.x and beyond. 

 
 [[/images/aodbcst.gif|]]

This driver is available for Windows, Linux, Unix(most flavours) and Mac OS X Operating systems.
