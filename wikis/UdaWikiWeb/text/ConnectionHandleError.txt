===Can't obtain connection handle===

"Can't obtain connection handle" messages are common and generic. Typically, "connection handle" errors appear when incorrect values are passed in ODBC DSN's or JDBC connection strings. 

If you receive a "connection handle" error, take note of any errors which accompany it. Then, search the Knowledgebase for these errors. If no errors accompany " Can't obtain connection handle," review <nop>OpenLink's ODBC DSN or JDBC URL configuration documentation.
