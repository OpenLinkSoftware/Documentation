
= doc.FAQ.CannotDownloadToSpecificMachine

%TOC%

   
Network or security policies may restrict a user's ability to download software onto a machine. These users should take the following action: 

*Identify a machine that is accessible to the Internet and the internal network.  
*Download [[OpenLink]]  software onto this machine.  
*Use ftp to transfer files to the appropriate destination.  
Alternatively users may: 

*Identify a machine that is accessible to the Internet.  
*Download [[OpenLink]]  software onto this machine.  
*Save [[OpenLink]]  software to disk or CD-ROM.  
*Transform the disk or CD-ROM to the appropriate machine.  
*Load the software from the disk or CD-ROM.
