===Conductor===

The Conductor is a deprecated, graphical user interface, which ships with older Virtuoso products. The Conductor is the prime administrative tool for Virtuoso 1.x. However, it has been replaced in subsequent releases. This Conductor allows users to create DSN's using <nop>OpenLink, native, or third-party drivers.
