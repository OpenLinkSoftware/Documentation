==FBS==

FBS (<nop>FetchBufferSize) is a UDBC connect string parameter. The value passed by FBS represents the number of rows of resultset data which you want fetched before data is returned to your client. A high FBS reduces the need to traverse back and forth across the network. However, a high FBS will result in longer intervals between the arrival of data. 

See Also: /FBS, Fetch Buffer Size, Row Buffer Size
