==Gathering Additional Evidence==
===Installation Shell Script===
The <nop>OpenLink installation shell script is used to extract and install Linux and Unix Multi-Tier software. This file is named install.sh. It should appear in the root directory of your <nop>OpenLink installation. Please copy this file and forward it to <nop>OpenLink Product Support.
