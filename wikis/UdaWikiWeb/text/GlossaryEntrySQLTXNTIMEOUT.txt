==SQL_TXN_TIMEOUT==

The SQL_TXN_TIMEOUT parameter appears in the [Client] section of your virtuoso.ini file. SQL_TXN_TIMEOUT is an ODBC extension which allows you to set a maximum duration for transactions. Be advised, if SQL_TXN_TIMEOUT is set to 0, no limit will be imposed.
