===Database Has the Wrong Version Number===

If you receive "Database has the wrong version number" error messages, you will need to [[RelinkMulti-TierDatabaseAgents|relink your database agent]] or switch to [[ServerAgentAdminProgress|sockets-based connections]].
    
===Evidence===

* Request Broker Log 
* Database Agent Version String 
* DBMS Version
