==INFORMIXSERVER==

This INFORMIXSERVER environmental variable is set in the [Environment INFORMIXX] section of your <nop>OpenLink Session Rules Book. This variable must pass the name of your Informix server. A corresponding variable should be found in the .profile of your Informix user. 

Note: This variable may be set using the Administrative Assistant's Database Agent Settings Form or Wizard.
