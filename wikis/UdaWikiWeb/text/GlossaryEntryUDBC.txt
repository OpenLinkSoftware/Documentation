==UDBC==

UDBC is <nop>OpenLink's deprecated, proprietary data access mechanism. It denotes "Universal Database Connectivity," and it provides connectivity by linking client applications directly to data access drivers. Thereby, dependence on a driver manager was bypassed, when no such manager existed on Unix platforms.

The iODBC driver manager project has removed the need for UDBC.
