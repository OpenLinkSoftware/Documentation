==Program==

The Program parameter occurs in the [generic_xx] sections of your <nop>OpenLink Session Rules Book.   
It corresponds to the Executable Name field in both the Administrative Assistant's Database Agent Settings Form and Wizard. This parameter specifies which agent executable is to be spawned for a given [generic_xx] section. 

Example: 

{{{
Program=pro90b_sv 
}}}

See Also: Executable Name
