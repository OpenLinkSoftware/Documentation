= Error Message: {{{Unknown table name <name>_0.(200)}}} =

"{{{Unknown table name <name>_0.(200)}}}" messages are associated with Microsoft Query and the Microsoft Excel Wizard. 

If you experience this error, [[ConfigureProgressTableview|implement Tableview]] and retest. 

(Note: Tableview is a two part procedure. You must first generate tableview {{{.dat}}} and {{{.pf}}} files. Then, you must configure the data access components to use these files.)

If the problem persists, [[http://support.openlinksw.com/support/|contact Product Support]].
