==<nop>MaxConnections==

The <nop>MaxConnections parameter occurs in the [Request Broker] section of the <nop>OpenLink Session   
Rules Book. If the value passed to <nop>MaxConnections is greater than 0, the Request Broker will interpret the value as the maximum number of connections which it may accept from clients, at a given time. 

Example: 

{{{
MaxConnections = 100 
}}}
