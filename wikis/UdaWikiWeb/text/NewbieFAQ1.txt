==Beginner UDA Questions==

===What do your drivers do?===

<nop>OpenLink's Data Access drivers enable users to establish ODBC and JDBC connections between applications and databases. Applications use these connections to execute SQL queries against the data contained within the databases. Typical SQL queries select, insert, update, or delete data. Other queries add, modify, or delete the tables that contain the data.
