===Access -7716===


-7716 is a Microsoft Access "unknown error" message. It does not have a specific meaning. To diagnose the problem, create a client-side ODBC trace and forward it to <nop>OpenLink Product Support.
    
===Evidence===

* ODBC Client Trace
* Request Broker Log 
* Windows Driver Manager Version
