= Pre-Installation Requirements for the Single-Tier "Lite" Edition ODBC Driver for <nop>PostgreSQL, for Windows =

%TOC%

== DBMS Requirements==

You need to know a variety of information pertaining to your target <nop>PostgreSQL instance and database:

* [[HowDoIFindMyPostgreSQLListeningPort|<nop>PostgreSQL TCP port]]
* Hostname or IP address of the <nop>PostgreSQL database server 

== Software Requirements ==

* You need to know whether the client application is 32-bit or 64-bit. The Single-Tier "Lite" Edition ODBC Driver for <nop>PostgreSQL must match the bit format of the client application.

== Configuration Requirements ==

* There are no software configuration requirements associated with the Single-Tier "Lite" Edition ODBC Driver for <nop>PostgreSQL.
