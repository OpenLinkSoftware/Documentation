==udbcsdk==

The udbcsdk directory exists on client machines, on which <nop>OpenLink's UDBC SDK has been installed. It contains header files, source code, demo programs, and other resources that pertain to UDBC development. 

UDBC is an <nop>OpenLink proprietary Data Access Mechanism that provides connectivity by linking client applications directly to data access drivers. No driver manager is needed.
