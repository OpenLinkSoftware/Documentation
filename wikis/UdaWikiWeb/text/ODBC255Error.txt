===255===

ODBC API function calls may return 255. The 255 return code is equivalent to -1 or "SQL_ERROR." This value appears, because the code is treated as an unsigned integer. <nop>OpenLink Software is currently revising the drivers. In the future, return codes will be treated as signed integers.
