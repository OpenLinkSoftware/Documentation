==SAP Registration==

SAP Registration The SAP Registration checkbox appears on the Edit Request Broker's Parameters Form. This form appears in the Server Components Administration{{{=}}}>Request Broker Administration menu in the <nop>OpenLink Administrative Assistant. If this box is checked, the Request Broker will forward SAP registration packets which contain the name OPENLINK. Local file servers and applications which listen for SAP packets will recognize the packets and become aware of the Request Broker's presence. 

See Also: <nop>SAPBroadcast, SAP Service Name, SAP Service Type, SPX
