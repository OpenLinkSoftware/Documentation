===VJ CLASSPATH too long to be changed===

"VJ CLASSPATH" errors occur when you install Single-Tier drivers deep within your computer's directory structure. For example:

{{{
C:\Program Files\Drivers\ODBC\OpenLink\Version 4.1\Lite32
}}}

To resolve this problem, install your drivers in a higher-level directory. For example:

{{{
C:\Program Files\Lite32
}}}
