==Keep Alive==

The Keep Alive Packets checkbox appears in the Administrative Assistant's Edit Request Broker's Parameters Form. If this box is checked, the Request Broker will forward ICMP echo reply packets to each connected client. If the client does not reply to two successive packets, the Broker will move to terminate that client's connections.
