==ORACLE_SID==

The ORACLE_SID environmental variable is set in the [Environment ORACLE] section of the <nop>OpenLink Session Rules Book. It passes the ORACLE_SID to which one wants to connect. 

Note: This variable may be set using the Administrative Assistant's Database Agent Settings Form or Wizard.
