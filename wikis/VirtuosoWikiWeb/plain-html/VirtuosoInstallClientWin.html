Installation


  1. The Client Connectivity Suite installer ships as a .msi file:





  2. Double click on the installer icon to commence the installation:





  3. Read and accept the license agreement:





  4. Select the type of installation to perform -- Typical, Complete, or Custom.





  5. Typical and Complete installations go to the default location, C:\Program Files\OpenLink Software\Virtuoso\. For Custom installation, you may choose the installation directory or accept the default:





  6. Typical and Complete installations include drivers and providers for all supported client mechanisms. For Custom installation, you may specify a more limited selection:





  7. Click the install button to commence the installation:





  8. The installation progress is displayed:





  9. The installation is now complete.
