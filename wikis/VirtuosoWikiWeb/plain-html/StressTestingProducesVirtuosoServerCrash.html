Stress Testing Produces Virtuoso Server Crash

If you experience a Virtuoso server crash, contact OpenLink Product Support and provide the following information:

1. Hardware and operating system details for the machine hosting the Virtuoso Universal Server.
2 A copy of the virtuoso.ini file.
3. A description of load including sample queries, where possible.
4. The output of the status() call made under less than critical load.
