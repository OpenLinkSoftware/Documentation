Is there a way to configure the display width of isql output?

Q: Can I set the display width for the isql demo application? My result set wraps around before the actual end of the line making it very hard to read.

A: There's no such configuration parameter for isql. OpenLink staff have used text consoles provided by Linux, KDE Konsole, OSX terminal, Cygwin and have not seen
any problems with isql display width.
