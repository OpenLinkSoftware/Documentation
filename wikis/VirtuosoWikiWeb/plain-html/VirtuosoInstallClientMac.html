Installing the Virtuoso Client Connectivity Suite on Mac OS X

  1. The Client Connectivity Suite installer ships as a .dmg file:





  2. Open the Virtuoso Disk Image file by double clicking the DMG file in the Finder windows or using the 'open' command from a Terminal window. This mounts the Disk Image containing the actual .mpkg installer:





  3. The installer will load the Introduction page:





  4. Review the readme page for any important information concerning the product installation:





  5. Read the license agreement and 'agree' to the terms:










  6. Select the Destination volume the software is to be installed on:





  7. The software must be installed by a user with Administrative privileges on the machine:





  8. The software components will be installed.
