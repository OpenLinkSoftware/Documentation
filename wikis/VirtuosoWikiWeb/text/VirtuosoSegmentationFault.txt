===Segmentation Fault===

If Virtuoso produces segmentation fault errors, obtain the following resources and forward it on to <nop>OpenLink Product Support:

1. A Virtuoso server version string. 

2. State whether Virtuoso is Open Source (VOS) or the commercial edition.

3. An strace log from an instance when the segmentation fault is occurring.

4. gdb stack trace output, if any cores are being produced.

5. A clientside ODBC trace against any query that may be associated with the error.

6. A code sample when PHP, PERL, or another scripting language is associated with the problem.
