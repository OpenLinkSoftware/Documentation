=== Box length too large 
 
This problem is associated with the older version of Virtuoso's JDBC drivers. Use the following command to return your driver version: 

{{{
java virtuoso.jdbc3.Driver 
}}}

For example: 

{{{
$ java virtuoso.jdbc3.Driver OpenLink Virtuoso(TM) Driver for JDBC(TM) Version 3.x [Build 3.25] 
}}}

Then, contact OpenLink Product Support to see if a newer version is available.
