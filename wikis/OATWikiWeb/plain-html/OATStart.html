Requirements

  • Web Server that optionally supports WebDAV?
  • XMLA Data Provider module (optional as this is only required to SQL Data Access over XMLA)

Installation Steps

  • Extract ZIP archive components to a physical directory that is mapped to a logical/virtual directory by your Web Server
  • Edit the "preferences.js" file so that it points to the logical/virtual directory created in step 1
