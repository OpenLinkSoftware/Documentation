iSPARQL Querying Tutorial executed against Virtuoso Northwind database

In this tutorial we will show how to execute a SPARQL query against the Virtuoso Northwind database.

  • Create Graph model, generate query, and execute

Create Graph model, generate query, and execute

  1. Go to iSPARQL demo page.
  2. Click "Ok" in the shown Login form. By default should be shown user name demo and password demo.



  3. As result will be shown the iSPARQL home page, with opened tab "QBE" and default graph elements will be shown in the Canvas. Click the "Clear Pane" icon form the toolbar in order to empty the Canvas.



  4. Change the "Data Source URI" field value from the QBE tab to: http://demo.openlinksw.com/Northwind as this is the Graph URL we are going to use in our tutorial.



  5. Now we will import the Schema that contains the Terms to be used in the our Graph Pattern. From the Schemas window enter in the input field the schema URL: http://www.openlinksw.com/schemas/northwind and click the "Import" button.



  6. As result in the Schemas window under "unbound" will appear new item http://www.openlinksw.com/schemas/northwind#. Click its "+" icon in order to expand it.
  7. Click the "+" icon for the Concepts Node in order to expand it.



  8. Now scroll down the shown list and drag-and-drop the Subject Node "Order" on to the QBE working Canvas.
