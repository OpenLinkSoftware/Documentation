---++ OAT API Index

   * [[OATDOCajax2][ajax2.js]]  - Provides basic Asynchronous XML call (AJAX) routines.
   * [[OATDOCanchor][anchor.js]] - Enhanced Anchor tag.
   * [[OATDOCanimation][animation.js]] - Implements basic animation framework, as well as some pre-defined animation effects.
   * [[OATDOCbarchart][barchart.js]]  - Library for drawing Bar charts, using HTML DOM.
   * [[OATDOCbezier][bezier.js]]  - Provides two mathematical functions for drawing Bezier curves.
   * [[OATDOCcalendar][calendar.js]]  - Displays a calendar for date picking.
   * [[OATDOCcolor][color.js]]  - Displays a color picker.
   * [[OATDOCcombobox][combobox.js]]  - Creates a combobox, an enhanced select which allows richer options.
   * [[OATDOCcombobutton][combobutton.js]]  - Creates a multifunctional button, which enables user to specify a callback action to be performed when it is clicked.
   * [[OATDOCcombolist][combolist.js]]  - Creates a combolist, select with additional possibility of direct user input.
   * [[OATDOCconnection][connection.js]]  - Connection object for data access.
   * [[OATDOCcrypto][crypto.js]]  - Provides cryptographic (hashing) functions.
   * [[OATDOCdatasource][datasource.js]]  - An Ajax Database Connectivity (ADBC) layer.
   * [[OATDOCdav][dav.js]]  - Displays a WebDavBrowser window for file picking/opening/saving.
   * [[OATDOCdialog][dialog.js]]  - Simple wrapper for ok/cancel dialog, that can contain any data.
   * [[OATDOCdimmer][dimmer.js]]  - Shows an object while dimming others, i.e., a Lightbox effect.
   * [[OATDOCdock][dock.js]]  - Creates a draggable system of docking elements, grouped into columns.
   * [[OATDOCdrag][drag.js]]  - Provides dragging (moving) capability for any DOM node.
   * [[OATDOCfisheye][fisheye.js]]  - Creates Mac-like dock.
   * [[OATDOCfresnel][fresnel.js]]  - Fresnel Lens RDF Processor.
   * [[OATDOCghostdrag][ghostdrag.js]]  - Provides drag'n'drop functionality.
   * [[OATDOCgraphsvg][graphsvg.js]]  - SVG-based RDF Graph viewer.
   * [[OATDOCgrid][grid.js]]  - Advanced grid (table) control.
   * [[OATDOCinstant][instant.js]]  - Provides pop-up functionality to any element.
   * [[OATDOCjson][json.js]]  - Implements two crucial JSON functions - parse and stringify.
   * [[OATDOCkeyboard][keyboard.js]]  - Keyboard events and shortcut handling.
   * [[OATDOClayers][layers.js]]  - Generic z-index manager.
   * [[OATDOClinechart][linechart.js]]  - Library for drawing Line charts, using SVG.
   * [[OATDOCloader][loader.js]]  - This is the basic library. Can be considered a toolkit on its own, as it provides many useful functions and shortcuts.
   * [[OATDOCmap][map.js]]  - Abstract API atop various mapping providers.
   * [[OATDOCmenu][menu.js]]  - Pop-up menu control.
   * [[OATDOCn3][n3.js]]  - N3/Turtle parser.
   * [[OATDOCnotify][notify.js]]  - Provides API for informative notification boxes.
   * [[OATDOCpanelbar][panelbar.js]]  - Panelbar widget (like mozilla sidebar).
   * [[OATDOCpiechart][piechart.js]]  - Library for drawing Pie charts, using SVG.
   * [[OATDOCpivot][pivot.js]]  - Advanced interactive Pivot table.
   * [[OATDOCquickedit][quickedit.js]]  - Provides edit-in-place functionality.
   * [[OATDOCrdf][rdf.js]]  - RDF/XML parser.
   * [[OATDOCrdfbrowser][rdfbrowser.js]]  - Multi-purpose RDF Browser.
   * [[OATDOCrdfstore][rdfstore.js]]  - Client-side RDF store.
   * [[OATDOCrdftabs][rdftabs.js]]  - Tabs API for OAT RDF applications.
   * [[OATDOCresize][resize.js]]  - Provides resizing capability for any DOM node.
   * [[OATDOCrotator][rotator.js]]  - Creates a set of 'rotating' elements, which are cyclically displayed in an infinite fashion (1, 2, 3, 1, 2 ...).
   * [[OATDOCrssreader][rssreader.js]]  - RSS Feed reader supporting both RSS and RDF feeds.
   * [[OATDOCsimplefx][simplefx.js]]  - Simple eye-candy effects: shadows, rounding etc.
   * [[OATDOCsoap][soap.js]]  - Provides basic asynchronous XML call (AJAX) routines for use with SOAP requests.
   * [[OATDOCsparqline][sparqline.js]]  - Library for drawing Sparklines using SVG.
   * [[OATDOCsqlquery][sqlquery.js]]  - SQL Query manipulation library. This is an incomplete library, thus without doc.
   * [[OATDOCstatistics][statistics.js]]  - Elementary statistic functions.
   * [[OATDOCsvg][svg.js]]  - SVG helper provides simplified way of SVG elements creation.
   * [[OATDOCtab][tab.js]]  - Provides tab functionality.
   * [[OATDOCtagcloud][tagcloud.js]]  - Displays a "Tag Cloud", visually enhanced set of links.
   * [[OATDOCticker][ticker.js]]  - Slowly appearing and disappearing text.
   * [[OATDOCtimeline][timeline.js]]  - Draggable timeline.
   * [[OATDOCtoolbar][toolbar.js]]  - Basic Toolbar control.
   * [[OATDOCtree][tree.js]]  - Convertsultag into interactive tree.
   * [[OATDOCupload][upload.js]]  - Creates form with variable number of upload slots.
   * [[OATDOCvalidation][validation.js]]  - Creates restricted input controls.
   * [[OATDOCwin][win.js]]  - Creates windows.
   * [[OATDOCws][ws.js]]  - Web Services (WSDL) interface.
   * [[OATDOCxml][xml.js]]  - Parses a string of XML data and returns a document object.
   * [[OATDOCxmla][xmla.js]] - XML for Analysis (XMLA) methods.
