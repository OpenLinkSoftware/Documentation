---++ Functions

---+++ OAT.Statistics.sum(array)

Sums all values from <tt>array</tt>.

---+++ OAT.Statistics.product(array)

Gives product of all values from <tt>array</tt>.

---+++ OAT.Statistics.amean(array)

Gives arithmetic mean of all values from <tt>array</tt>.

---+++ OAT.Statistics.max(array)

Returns highest value from <tt>array</tt>.

---+++ OAT.Statistics.min(array)

Returns lowest value from <tt>array</tt>.

---+++ OAT.Statistics.distinct(array)

Counts distinct values in <tt>array</tt>.

---+++ OAT.Statistics.variance(array)

Returns variance of <tt>array</tt>.

---+++ OAT.Statistics.deviation(array)

Returns standard deviation of <tt>array</tt>.

---+++ OAT.Statistics.median(array)

Returns standard median of <tt>array</tt>.

---+++ OAT.Statistics.mode(array)

Returns standard mode of <tt>array<tt>.
