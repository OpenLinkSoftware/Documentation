Calendar OAuth Ubiquity Authentication Example

The following example demonstrates oauth authentication getting the OAuth SID value for ex. for existing Calendar instance "mycalendar" with owner user demo at the OpenLink Demo server.

In order to execute correctly the example, you need to have OAuth Generate Key for the Calendar "mycalendar" instance at OpenLink Demo server. More information and simple scenario how to be done you can find here.

  1. Execute the command: 

    ods-oauth-host http://demo.openlinksw.com/OAuth

  2. Execute the command: 

    ods-set-mode oauth

  3. Go to http://demo.openlinksw.com/ods/oauth_sid.vsp
  4. In the shown form:
    1. Enter for user demo
    2. Choose the application "mycalendar".
    3. Click the "Execute" button.
  5. Enter for user demo its password and click the "Login" button.
  6. Click the "Authorize" button.
  7. Copy the obtains OAuth SID value for ex.: dca421eefa2d578e19c0b2f987e7a9ca
  8. Execute the following command: 

    ods-set-calendar-oauth dca421eefa2d578e19c0b2f987e7a9ca
