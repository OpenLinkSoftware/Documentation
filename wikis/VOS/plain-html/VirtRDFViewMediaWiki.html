MediaWiki RDF Views

Each MediaWiki? installation includes a demonstration database that includes a management system schema and instance data. This sample database provides an easy route for introducing MediaWiki? users and developers to the virtues of RDF based Linked Data Views built atop existing MediaWiki? relational databases. In the sections that follow, we include links for the following:
