Example Usage of ODS Briefcase Controller for Get Collection Details

Note: The Consumer key for a given ODS Briefcase instance can be obtained following these steps.

  1. At http://demo.openlinksw.com/ods/oauth_test.vsp enter:
    1. Server: http://demo.openlinksw.com/ods/api/
    2. Consumer Key: f3da6bd04ad341f9d79fddc7ea224f50576e74a2
    3. API method: briefcase.collection.info
    4. API parameters: path=/DAV/home/demo/mytest
    5. Query options: Generate URI and Submit
    6. Log in when prompted as user demo and authorize
  2. As result the Response Body should be: 

    <item path="/DAV/home/demo/mytest/">
  <name>mytest</name>
  <mimeType>dav/unix-directory</mimeType>
  <owner>demo</owner>
  <group>demo</group>
  <permissions>rw-r--r--</permissions>
  <modification>2009-11-30</modification>
  <creation>2009-11-30</creation>
  <properties></properties>
  <shares></shares>
</item>
