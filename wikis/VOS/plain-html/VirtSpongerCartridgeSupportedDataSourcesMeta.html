Meta Cartridges

MetaCartridges supplement and enhance RDF data in a pre-existing graph provided by an extractor cartridge, either by local processing or by submitting a subset of the graph data (such as a list of entity names, or document content) to an upstream service for further processing.

Default Sponger behavior is for all installed Meta Cartridges to be brought to bear on all submitted Resources, except where an API key or similar is required.
