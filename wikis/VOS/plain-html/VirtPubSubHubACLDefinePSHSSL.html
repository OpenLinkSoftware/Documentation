Define the PSH-SSL ACL list

Using Conductor UI we define Access control lists for FOAF+SSL authentication:

  1. Go to Conductor ->System Admin -> Security -> Access Control



  2. Enter "PSH-SSL"



  3. Click "Add New ACL"



  4. Click "Add New Rule"



  5. Enter in the shown form:
    • Filter: your FOAF+SSL WebID?, for ex.: http://webid.myxwiki.org/xwiki/bin/view/XWiki/rumito#me
    • Access: Allow
    • Destination: *
    • Object, Mode, Rate and Limit: 0



  6. Click "Add"
  7. As result the rule will be created.



  8. Click "Add New Rule"
  9. Enter in the shown form:
    • Filter: *
    • Access: Deny
    • Destination: *
    • Object, Mode, Rate and Limit: 0



  10. Click "Add"
  11. As result the rule will be created.



  12. Click "Apply"
  13. As result PSH-SSL ACL with its rules will be created:
