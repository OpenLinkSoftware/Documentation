DET, or "DAV Extension Type"

DET is short for DAV Extension Type. DETs are a means of programmatically creating WebDAV collections and resources from DBMS data (Relational, Graph, or otherwise) on the fly (i.e., actual content is computed at run time).

This is the mechanism used by ODS Briefcase to expose data from the following SQL-based applications as URL locatable resources, for example:

  1. ODS Weblog
  2. ODS Feed Manager
  3. ODS Wiki
  4. ODS Gallery
  5. ODS Bookmark Manager
  6. ODS Mail

This is also the mechanism that enables URI referencing of SQL Data via WebDAV for RDF Triple Statement construction.
