SPARQL Endpoint DET Configuration Guide: set the SQL user account to be DAV enabled Using Conductor UI

  1. Make sure the steps from the Secure SPARQL Endpoint Guide using WebID Protocol are performed.
  2. Go to http://cname/conductor
  3. Enter dba credentials
  4. Go to System Admin -> User Accounts



  5. Locate user "SPUID1" and click the"Edit" link from the very last right column.



  6. In the shown form change "User type" to "SQL/ODBC and WebDAV":



  7. Click "Save"
