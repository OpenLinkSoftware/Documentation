Creating PSH Virtual Directory

  1. Go to Conductor-> Web Application Server -> Virtual Domains & Directories
  2. If there is no /psh folder defined for your HTTPS Listener, click the "New Directory" link in order to create it:



  3. In the shown form leave selected "None" and click "Next"



  4. In the "HTTP Virtual Directory" form enter:
    1. Path: /psh
    2. Hatch "Physical path is a WebDAV repository"
    3. Physical path: /DAV/VAD/pubsubhub/
    4. Default page: index.vsp



  5. Click "Save Changes"
  6. As result the Virtual Directory will be created.
