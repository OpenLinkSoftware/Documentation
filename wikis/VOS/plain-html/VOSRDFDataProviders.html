Virtuoso RDF Data Providers

We have produced Data Model Storage Providers for the Sesame, Redland, and Jena Frameworks. These providers facilitate the native persistence of RDF data via Virtuoso's Quad Store. These providers give better performance than those that persist RDF in SQL form, since they remove the data-model impedance overhead between the RDF graph model and the relational SQL model.

  • Virtuoso Jena RDF Storage Provider
  • Virtuoso Eclipse RDF4J & OpenRDF Sesame RDF Storage Providers
  • Virtuoso Redland RDF Storage Provider
