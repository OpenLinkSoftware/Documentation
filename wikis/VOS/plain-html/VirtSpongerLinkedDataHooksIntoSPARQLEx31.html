Example with pragma sql:big-data-const

The following example demonstrates usage of the sql:big-data-const pragma.


DEFINE sql:big-data-const 0
DEFINE sql:signal-void-variables 1 
SELECT DISTINCT ?g
WHERE
   {
      GRAPH ?g {?s a ?t}
   }
LIMIT 10
