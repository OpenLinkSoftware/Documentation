Building the Python hosting plugin for the Virtuoso Server

  • Prerequisites
  • Building the plugin

Prerequisites

  • The Python hosting plugin requires Python 2.4 or later.

Building the plugin

To build the Python plugin along with the Virtuoso server binary, add the following when you configure the Virtuoso build -- 


--enable-python
