---+Example with pragma output:format


The following example demonstrates usage of the <code> output:format </code> pragma.


<verbatim>
DEFINE output:format "TTL" 
SELECT * 
WHERE 
  {
    graph ?g { ?s ?p ?o }
  } 
LIMIT 10
</verbatim>
