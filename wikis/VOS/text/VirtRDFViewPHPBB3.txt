---++<nowiki>phpBB</nowiki> 3 RDF Views

Each phpBB 3 installation includes a demonstration database that includes a management system schema and instance data. This sample database provides an easy route for introducing phpBB 3 users and developers to the virtues of RDF based Linked Data Views built atop existing phpBB 3 relational databases. In the sections that follow, we include links for the following:

   1 Linked Data View Creation Script
   1 Live Examples of Linked Data Views built using the script above.
