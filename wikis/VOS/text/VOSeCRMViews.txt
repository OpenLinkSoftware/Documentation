---++eCRM Views

Each eCRM installation includes a demonstration database that includes a custom relationship management system schema and instance data. This sample database provides an easy route for introducing ODS eCRM users and developers to the virtues of RDF based Linked Data Views built atop existing eCRM relational databases. In the sections that follow, we include links for the following:

   1 Linked Data View Creation Script
   1 Ontology Generation
   1 Deployment of the Ontology and Instance Data to the Web of Linked Data
   1 Live Examples of Linked Data Views built using the script above.
