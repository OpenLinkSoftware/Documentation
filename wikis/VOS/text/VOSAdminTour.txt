---+ Admin Interface Tour


Open the interface by connecting to the Virtuoso Conductor at the default listener of the demo database, normally <code>[[http://localhost:8890/conductor/][http://localhost:8890/conductor/]]</code>.

The following functionality areas may be interesting:

   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#querytools][Interactive SQL]] - perform ad hoc queries;
   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#webdavadmin][WebDAV Administration]] - view DAV resources, manage their access rights and content;
   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#admui.internetdomains][Internet Domains Administration]] - Create and manage HTTP virtual directories;
   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#dbadmin][Database Administration]] -  Create and manage 
users and group accounts, manage database, edit triggers, stored procedures tables and views, manage external data sources, view statistics, use profiling and many others;
   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#replicationandsynchronization][Replication & Synchronization]] - Create and manage replications from type "Basic", "Incremental", "Bidirectional Snapshot" and "Transactional";
   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#rdfadm][Linked Data Administration]] - Perform statistics and manage graphs, import schemas and define namespaces, generated Linked Data Views and upload to the Quad Store;
   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#admui.xmlservices][XML Services]] - Browse through ready-made SQLX reports on data in the demo database;
   * [[http://docs.openlinksw.com/virtuoso/htmlconductorbar.html#admui.webservices][Web Services]] - See what SOAP end points are defined and what procedures these expose.
