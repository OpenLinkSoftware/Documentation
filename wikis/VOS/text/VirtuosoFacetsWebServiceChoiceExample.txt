---+ Choice of Labels Example

   1 Go to <code><nowiki>http://lod.openlinksw.com/sparql</nowiki></code>
   1 Enter the following query in the text area:
<verbatim>
DEFINE  output:valmode   'LONG' 
DEFINE  input:inference  'facets' 
SELECT * 
  WHERE
    { 
      ?s  virtrdf:label  ?o 
    }
</verbatim>
