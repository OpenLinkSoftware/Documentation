---+Example with pragma sql:describe-mode


The following example demonstrates usage of the <code> sql:describe-mode </code> pragma.


<verbatim>
DEFINE sql:describe-mode "CBD"
PREFIX mo: <http://purl.org/ontology/mo/>
DESCRIBE ?s 
FROM <http://www.bbc.co.uk/music/> 
WHERE 
  { 
        ?s a mo:SoloMusicArtist ;
                mo:image ?image ;
    mo:musicbrainz ?musicbrainz ;
                foaf:page ?href ;
                foaf:name ?name ;
         rdfs:comment ?comments .
    OPTIONAL { ?s mo:wikipedia ?wikipedia } .
    OPTIONAL { ?s mo:biography ?bio } .
    OPTIONAL { ?s mo:myspace ?myspace } .
    OPTIONAL { ?s mo:imdb ?imdb } .
}

LIMIT 100
</verbatim>
