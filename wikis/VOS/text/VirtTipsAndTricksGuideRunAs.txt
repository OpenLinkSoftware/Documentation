---++How can I specify OS user name to which the Virtuoso server will switch after opening the listen ports?

You can specify the OS user name to which the Virtuoso server will switch after opening the listen ports, by setting the <b>RunAs</b> parameter in the Virtuoso ini file, in the  <b>Parameters</b> section.
