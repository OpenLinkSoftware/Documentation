---+++Feed Manager Example for Extracting Metadata and DET based resources generation

Here is sample scenario:

   1 Go to http://&lt;cname&gt;/ods
   1 Log in as user
   1 Go to Feed Manager -> Administration and subscribe for feed, for ex. <nowiki>http://www.openlinksw.com/blog/~kidehen/gems/rss.xml</nowiki>:
%BR%%BR%<img src="%ATTACHURLPATH%/b40.png" />%BR%%BR%  
   1 As result you will be subscribed for the feed in question
%BR%%BR%<img src="%ATTACHURLPATH%/b41.png" />%BR%%BR%  
   1 Go to Briefcase path: DAV/home/user-name/Items/Syndicated Content 
%BR%%BR%<img src="%ATTACHURLPATH%/b42.png" />%BR%%BR%  
   1 View the content of the shown folders, for ex. view the "title" folder content:
%BR%%BR%<img src="%ATTACHURLPATH%/b43.png" />%BR%%BR%  
   1 As result will be shown in the list a folder with name "Kingsley Idehen's Blog Data Space", which is actually the name of the blog with the given RSS Feed file from above.
   1 Now go to DAV/home/user-name/Feed Subscriptions
%BR%%BR%<img src="%ATTACHURLPATH%/b43a.png" />%BR%%BR%  
      * Note the folder "Feed Subscriptions" will be created once your user creates Feed Manager instance.
   1 As result will be shown list of you Feed Manager subscribed for channels.
   1 Click on the channels link
%BR%%BR%<img src="%ATTACHURLPATH%/b43b.png" />%BR%%BR%  
   1 As result will be shown list of the subscription feeds.
