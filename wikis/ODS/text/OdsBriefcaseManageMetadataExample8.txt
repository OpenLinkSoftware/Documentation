---+++Wiki Example for Extracting Metadata and DET based resources generation

Here is sample scenario:

   1 Go to http://&lt;cname&gt;/ods
   1 Log in as user
   1 Go to Wiki and create article for ex. "MyTest":
%BR%%BR%<img src="%ATTACHURLPATH%/b44.png" />%BR%%BR%  
   1 Enter some body text and click "Save and release lock"
%BR%%BR%<img src="%ATTACHURLPATH%/b45.png" />%BR%%BR%  
   1 Go to Briefcase path: DAV/home/user-name/Items/Wiki (OpenLink) 
%BR%%BR%<img src="%ATTACHURLPATH%/b46.png" />%BR%%BR%  
   1 View the content of the shown folders, for ex. view the "Title" folder content:
%BR%%BR%<img src="%ATTACHURLPATH%/b47.png" />%BR%%BR%  
   1 As result will be shown in the list a folder with name "MyTest", which is actually the name of article created from above.
   1 Now go to DAV/home/user-name/wiki
%BR%%BR%<img src="%ATTACHURLPATH%/b47a.png" />%BR%%BR%  
      * Note the folder "wiki" will be created once your user creates wiki instance.
   1 As result will be shown list of you Wiki instances
   1 Click on an wiki instance link
%BR%%BR%<img src="%ATTACHURLPATH%/b47b.png" />%BR%%BR%  
   1 As result will be shown list of the current articles for the wiki instance and also will be shown:
      * "attach" folder: contains information for the attached files
      * "VVC" folder: contains the different versions of the wiki articles.
