---++ OAuth Generate Keys for ODS Controllers (Web Services)

---+++ ODS Setup Application OAuth keys

   1. Login as user at ODS Data Space
   1. Go to Settings -> OAuth keys
%BR%%BR%<img src="%ATTACHURLPATH%/oa1.png" style="wikiautogen" width="600" />%BR%%BR%
   1. Select Application name from the drop-down list and click the "Generate Keys" button.
   1. The generated Token and Secret values for the selected application will be shown as a result.
%BR%%BR%<img src="%ATTACHURLPATH%/oa2.png" style="wikiautogen" width="600" />%BR%%BR%
