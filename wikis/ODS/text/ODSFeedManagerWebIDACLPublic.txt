---++ODS FeedManager WebID Protocol Sharing to Public

   1 [[ODSSetupSSL][Configure Virtuoso+ODS instance as an  X.509 Certificate Authority and HTTPS listener]]
   1 [[ODSGenerateX509Certificate][Generate a Personal HTTP based Identifier]] for fictitious Person Entity: John and then bind his personal Identifier to an X.509 Certificate (thereby giving him a WebID)
   1 Log into ODS-FeedManager as user "Demo" (a Person Entity)
   1 [[ODSFeedManagerShareSubscriptionToPublic][Share an ODS-FeedManager Subscription]] to Public.
   1 As user that has a valid WebID Protocol certificate: John, attempt to view the Subscription shared by Person Entity "Demo" by doing the following: 
      * Refresh browser;
      * Access the Feed Manager instance url: 
<verbatim>
https://host:ssl-port/dataspace/username/subscriptions/user-feedmanager-instance-name/
</verbatim>
      * For this example it could be: 
<verbatim>
https://ods-qa.openlinksw.com/dataspace/demo/subscriptions/Demo%20User%27s%20Feed%20Manager/
</verbatim>
      * When prompted by browser, select John's Certificate
