---++OpenLink Data Spaces (ODS) Adds support for [[ODSOpenID][OpenID]] and [[ODSYadis][Yadis]] Protocols

---+++Release 1.30.46

OpenLink Software are pleased to announce release  of OpenLink Data Spaces (ODS) version 1.30.46.

This release adds Client and Server side support for the OpenID for browser-based authentication and the Yadis authentication method discovery protocol to the Semantic Web Data Space Platform.
