---++Importing events and tasks in ODS Calendar

In ODS Calendar you can import your tasks and events using the Import feature from :
   * File
   * WebDAV
   * URL

Here is a sample scenario:
 
   1. Go to http://demo.openlinksw.com/ods
   1. Log in as demo with password demo
   1. Go to Calendar.
      * <img src="%ATTACHURLPATH%/ci1.png" style="wikiautogen"/>
   1. Go to Import/Export from the left vertical navigation and choose Import.
      * <img src="%ATTACHURLPATH%/ci2.png" style="wikiautogen"/>
   1. In the shown form:
      1. Select source type: File
      1. For Source name click the "Browse" button and select a .ics file form your local file system.
      1. Enter for Tags for ex.: ldp
      1. Click the "Import" button.
      * <img src="%ATTACHURLPATH%/ci3.png" style="wikiautogen"/>
   1. As result your iCalendar file content will be imported.
      * <img src="%ATTACHURLPATH%/ci4.png" style="wikiautogen"/>
