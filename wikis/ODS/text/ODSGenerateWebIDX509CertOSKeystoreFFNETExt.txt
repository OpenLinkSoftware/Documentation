---+Generate an X.509 Certificate (with a WebID watermark) that's managed by host operating system keystore: Firefox

   1 [[ODSSetupSSL][Set up an X.509 certificate issuer and HTTPS listener]].
   1 [[ODSX509GenerateIESetService][Set up X.509 Certificate Service]].
   1 Install the .NET extension for Firefox.
   1 Go to the ODS home page (local or remote) and register new user (or log in as existing ODS user).
%BR%%BR%<a href="%ATTACHURLPATH%/x13.png" target="_blank"><img src="%ATTACHURLPATH%/x13.png" width="600px" /></a>%BR%%BR%
   1 Post registration you end up with an ODS generated Person Entity Identifier (an HTTP URI that Identifies "You") that takes the form: 
<verbatim>
http://cname/dataspace/person/user-name#this
</verbatim>
%BR%%BR%<a href="%ATTACHURLPATH%/x14.png" target="_blank"><img src="%ATTACHURLPATH%/x14.png" width="600px" /></a>%BR%%BR%
   1 Edit your profile (Note: If this step is skipped, the resulting X.509 certificate will not have a human readable name):
%BR%%BR%<a href="%ATTACHURLPATH%/x15.png" target="_blank"><img src="%ATTACHURLPATH%/x15.png" width="600px" /></a>%BR%%BR%
   1 Go to Profile Edit -> "Security":
%BR%%BR%<a href="%ATTACHURLPATH%/x1.png" target="_blank"><img src="%ATTACHURLPATH%/x1.png" width="600px" /></a>%BR%%BR%
   1 Go to "Certificate Generator".
   1 Should be displayed the Virtuoso Certificate Generator Wizard:
%BR%%BR%<a href="%ATTACHURLPATH%/x2.png" target="_blank"><img src="%ATTACHURLPATH%/x2.png" width="600px" /></a>%BR%%BR%
%BR%%BR%<a href="%ATTACHURLPATH%/x3.png" target="_blank"><img src="%ATTACHURLPATH%/x3.png" width="600px" /></a>%BR%%BR%
   1 Click "Next".
   1 Specify Certificate type:
%BR%%BR%<a href="%ATTACHURLPATH%/x4.png" target="_blank"><img src="%ATTACHURLPATH%/x4.png" width="600px" /></a>%BR%%BR%
   1 Click "Next".
   1 Specify Certificate details:
%BR%%BR%<a href="%ATTACHURLPATH%/x5.png" target="_blank"><img src="%ATTACHURLPATH%/x5.png" width="600px" /></a>%BR%%BR%
   1 Click "Next".
   1 Specify Issuer:
%BR%%BR%<a href="%ATTACHURLPATH%/x6.png" target="_blank"><img src="%ATTACHURLPATH%/x6.png" width="600px" /></a>%BR%%BR%
   1 Click "Next".
   1 Select in the "Install or save" page option "Install the certificate in the Windows Certificate Store":
%BR%%BR%<a href="%ATTACHURLPATH%/x7.png" target="_blank"><img src="%ATTACHURLPATH%/x7.png" width="600px" /></a>%BR%%BR%
   1 Click "Next".
   1 Click "Copy" in order to copy the generated certificate and click "Exit":
%BR%%BR%<a href="%ATTACHURLPATH%/x8.png" target="_blank"><img src="%ATTACHURLPATH%/x8.png" width="600px" /></a>%BR%%BR%
   1 View the generated certificate saved in your Windows KeyStore:
      1. From <b>Run</b> run <code>mmc</code>.
      1. The Microsoft Management Console will pop up.
      1. Go to File ?> Add /Remove Snap-in.
      1. Go to Standalone(TAB) ?> Add ?> Certificates.
      1. Select Computer Account and click "Next".
      1. Select Local computer.
      1. Click Finish and Close.
   1 Go back to Security -> X.509 Certificates:
%BR%%BR%<a href="%ATTACHURLPATH%/x9.png" target="_blank"><img src="%ATTACHURLPATH%/x9.png" width="600px" /></a>%BR%%BR%
   1 Paste the copied generated certificate from above in the "X.509 Certificate" area:
%BR%%BR%<a href="%ATTACHURLPATH%/x10.png" target="_blank"><img src="%ATTACHURLPATH%/x10.png" width="600px" /></a>%BR%%BR%
   1 Note that you can also hatch the option "Enable Automatic WebID Login":
%BR%%BR%<a href="%ATTACHURLPATH%/x11.png" target="_blank"><img src="%ATTACHURLPATH%/x11.png" width="600px" /></a>%BR%%BR%
   1 Click the "Save Certificate" button which writes the Public Key Component of the generated Certificate to your FOAF based ODS hosted Profile data space:
%BR%%BR%<a href="%ATTACHURLPATH%/x12.png" target="_blank"><img src="%ATTACHURLPATH%/x12.png" width="600px" /></a>%BR%%BR%
   1 Log out and access for ex.: [[https://id.myopenlink.net/ods][https://id.myopenlink.net/ods]].
   1 Select when prompted from your browser, the certificate generated from above:
%BR%%BR%<a href="%ATTACHURLPATH%/x16.png" target="_blank"><img src="%ATTACHURLPATH%/x16.png" width="600px" /></a>%BR%%BR%
   1 Click "WebID Login":
%BR%%BR%<a href="%ATTACHURLPATH%/x17.png" target="_blank"><img src="%ATTACHURLPATH%/x17.png" width="600px" /></a>%BR%%BR%
   1 As result you should be successfully logged in:
%BR%%BR%<a href="%ATTACHURLPATH%/x18.png" target="_blank"><img src="%ATTACHURLPATH%/x18.png" width="600px" /></a>%BR%%BR%
   1 Now log out again and log in as another user, for ex. demo. 
   1 [[ODSBriefcaseWebIDPerson][Perform an WebID based ACL Sharing of a resource]] to the user with the generated from above X.509 hosted WebID.

Note: these steps apply to any ODS instance.
