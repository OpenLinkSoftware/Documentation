---++ODS Programmers Guide

---+++What

This article provides additional details on the ODS package and on the programming 
conventions followed in its writing. The additional details should help those who 
want to write procedures that directly manipulate the ODS data structures. Here 
also are included guides with installation steps, hints, and useful data-managing 
description code.

---+++Why

ODS Dataspaces provide a cost-effective route for generating Semantic Web Presence from Web 2.0 and traditional Web data-sources, by delivering an atomic data container for RDF Instance Data derived from data hosted in Blogs, Wikis, Shared Bookmark Services, Discussion Forums, Web File Servers, Photo Galleries, etc.
