Define the PSH ACL list

Using Conductor UI we define a PSH ACL list:

  1. Download and install the conductor_dav.vad and ods_framework_dav.vad packages.
  2. Go to Conductor ->System Admin -> Security -> Access Control -> PSH:



  3. Click "Add New Rule":



  4. In the presented form enter for ex.:
    • Filter: 127.0.0.1
    • Access: Allow
    • Destination: The Feed Resource URL you will subscribe for, for ex.: http://ods-qa.openlinksw.com/weblog/dav/dav-blog-0/gems/atom.xml
    • Object, Mode, Rate and Limit: 0



  5. Click "Add":



  6. Click "Apply":



  7. Click "Add New Rule"



  8. In the presented form enter for ex.:
    • Filter: The IP Address of your machine
    • Access: Allow
    • Destination: The Feed Resource URL you will subscribe for, for ex.: http://ods-qa.openlinksw.com/weblog/dav/dav-blog-0/gems/atom.xml
    • Object, Mode, Rate and Limit: 0



  9. Click "Add":



  10. Click "Apply":



    1 Click "Add New Rule":



  11. In the presented form enter for ex.:
    • Filter: *
    • Access: Deny
    • Destination: *
    • Object, Mode, Rate and Limit: 0



  12. Click "Add":



  13. Click "Apply":
