Sharing files and folders in ODS Briefcase

Any file or Folder can be shared to other members of ODS from Briefcase or non-members through WebDAV?. To share resource of folder, perform the following steps:

  1. Go to Briefcase
  2. Open the folder containing the resource (or sub-folder) you wish to share.
  3. Click the "Update properties" icon from column "Actions" for the resource or folder name.
  4. Go to tab "Sharing"
  5. Enter User or Group you want to share the resource to, the type of access (permissions) you want the resource to have
  6. Click the "Add" button.
  7. As result will be shown 1 new row in the sharings. Click the "Update" button in order to complete the sharing activity.
  8. Now let's check for user test1 what does it have in its Briefcase instance. Log in ODS as user test1
  9. Go to Briefcase
  10. As result will be shown the test1's Briefcase instance.
  11. Click the "Shared Resources" icon from the Briefcase Main Horizontal Navigation bar.
  12. As result will be shown list of shared to user test1 resources from other ODS users. Here is also the file we've shared above kidehen.vcf.
