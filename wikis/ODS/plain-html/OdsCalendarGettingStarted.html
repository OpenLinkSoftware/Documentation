ODS Calendar Getting Started

Prerequisites

  1. Virtuoso Server has been installed and is running
  2. ODS and other ODS Applications including Calendar have been successfully installed
  3. Register a new ODS account or login using your Virtuoso installers 'dba' or 'dav' accounts.

Step By Guide

  1. Using a web browser, login into you ODS user account or login as DAV or DBA by typing the following URL into your web browser: http://localhost:8890/ods/
  2. Select the Calendar tab from the ODS Bar.
  3. Select the "Create New" Link.
  4. Enter your Calendar name, URL and click the button labeled "Create and Launch Application", which will create a new Calendar instance.

Create Event

Once you have created an instance, you are ready to create and manage your calendar.

  1. Select the 'New Event' link form the left vertical navigation.
  2. Enter subject
  3. Select Duration
  4. Enter your description (optional)
  5. Enter location (optional)
  6. Set Repeat period (optional)
  7. Set range of Repeat (optional)
  8. Set Reminder (optional)
  9. Click "Create" button.

Create Task

  1. Go to tab "Tasks" from the Calendar Main Navigation bar.
  2. Click the button "New Task"
  3. Enter subject
  4. Specify Start Date (optional)
  5. Specify End Date (by default is set to current day)
  6. Set Priority within the list: highest, high, normal, low, lowest
  7. Set Status within the list: not started, in progress, completed, waiting,deferred
  8. Set % of complete
  9. Add description
  10. Click the button "Create".

Create Note

  1. Go to tab "Notes" from the Calendar Main Navigation bar.
  2. Click the button "New Note"
  3. Enter subject
  4. Add description
  5. Click the button "Create".

View Events

  1. Go to the tab "Events" from the Calendar Main Navigation bar.
  2. As result all your events will be shown in a list showing duration and subject of each event.

View Tasks

  1. Go to the tab "Tasks" from the Calendar Main Navigation bar.
  2. As result all your tasks will be shown in a list showing period of duration, subject, status and date of update for each task.

View Notes

  1. Go to the tab "Notes" from the Calendar Main Navigation bar.
  2. As result all your notes will be shown in a list showing subject and date of update for each note.

Calendar Page Overview

Check out some of Calendar other features as show below:

  1. Use Navigation toolbar to view events, task and notes by Day, Week and Month
  2. Export and import Calendar data
  3. Syndicate your calendar in various feed formats
  4. Search for events, tasks, notes
  5. Update preferences.
