Define PSH-SSL ACL list

Using Conductor UI we define Access control lists for WebID? authentication:

  1. Download and install the conductor_dav.vad and ods_framework_dav.vad packages.
  2. Go to Conductor ->System Admin -> Security -> Access Control -> PSH-SSL:



  3. Click "Add New Rule":



  4. In the presented form enter for ex.:
    • Filter: your WebID?, for ex.: http://webid.myxwiki.org/xwiki/bin/view/XWiki/demo#me
    • Access: Allow
    • Destination: *
    • Object, Mode, Rate and Limit: 0



  5. Click "Add":



  6. Click "Apply":



  7. Click "Add New Rule"



  8. In the presented form enter for ex.:
    • Filter: *
    • Access: Deny
    • Destination: *
    • Object, Mode, Rate and Limit: 0



  9. Click "Add":



  10. Click "Apply"
  11. As result PSH-SSL ACL with its rules will be created:
