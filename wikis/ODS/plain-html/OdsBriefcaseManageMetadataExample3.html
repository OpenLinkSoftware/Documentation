Graphic Image Example for Extracting Metadata and DET based resources generation

Here is sample scenario:

  1. Go to http://<cname>/ods
  2. Log in as user
  3. Go to Briefcase



  4. Upload an image file, for ex. kartina.jpg



  5. Now to view the retrieved metadata, click for the uploaded resource the "Update Properties" icon from the Action column.



  6. Go to "Metadata"



  7. Now to view the generated DET based resource, go to folder "Items" by clicking its name link.



  8. Go to "Graphics"
