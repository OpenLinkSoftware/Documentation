Discussions Controllers

Groups get

  • Description: get list of available groups
  • API name: ODS.ODS_API."discussion.groups.get"
  • Parameters: not such

Group get

  • Description: get group info by given group id
  • API name: ODS.ODS_API."discussion.group.get"
  • Parameters:
    • group_id

Group new

  • Description: creates new group
  • API name: ODS.ODS_API."discussion.group.new"
  • Parameters:
    • name
    • description

Group Remove

  • Description: removes group by given id
  • API name: ODS.ODS_API."discussion.group.remove"
  • Parameters:
    • group_id

Feed new

  • Description: created new feed for group
  • API name: procedure ODS.ODS_API."discussion.feed.new"
  • Parameters:
    • group_id
    • name

Feed remove

  • Description: removes feed
  • API name: ODS.ODS_API."discussion.feed.remove"
  • Parameters:
    • feed_id

Message new

  • Description: posts new message
  • API name: ODS.ODS_API."discussion.message.new"
  • Parameters:
    • group_id
    • subject
    • body

Message get

  • Description: gets message by given message id
  • API name: ODS.ODS_API."discussion.message.get"
  • Parameters:
    • message_id

Comment new

  • Description: creates new comment
  • API name: ODS.ODS_API."discussion.comment.new"
  • Parameters:
    • parent_id
    • subject
    • body

Comment get

  • Description: gets comment by given comment id
  • API name: ODS.ODS_API."discussion.comment.get"
  • Parameters:
    • comment_id
