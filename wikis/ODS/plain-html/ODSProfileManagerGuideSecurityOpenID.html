OpenID?

In the OpenID? tab, the ODS user can change his/her default OpenID? URI, which is respectively:

  • for a person: 

    http://<host>:<port>/dataspace/person/<user-name>#this

  • for an organization: 

    http://<host>:<port>/dataspace/organization/<organization-name>#this

  1. In the presented form, enter the new user's OpenID? URI.
    • Note: In order to restore the default OpenID? URI, empty the box.



  2. Click Save OpenID?.
