Virtuoso Tutorial RDF Views

Each Virtuoso Tutorial installation includes a demonstration database that includes a Virtuoso Tutorial management system schema and instance data. This sample database provides an easy route for introducing Virtuoso users and developers to the virtues of RDF based Linked Data Views built atop existing Virtuoso relational databases. In the sections that follow, we include links for the following:

  1. Linked Data View Creation Script
  2. Live Examples of Linked Data Views built using the script above.
