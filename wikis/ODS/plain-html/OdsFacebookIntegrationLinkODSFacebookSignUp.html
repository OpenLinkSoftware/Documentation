Link ODS user with Facebook account using the Sign Up Facebook feature

  1. Make sure the steps from the ODS Facebook Integration Administrators Guide are performed.
  2. Go to http://cname:port/ods .



  3. Click "Sign Up".



  4. Go to "Facebook" tab.



  5. Click the "Connect" button



  6. Enter in Facebook window form your Facebook credentials and click Login:



  7. Go to the Facebook tab



  8. Click "Sign Up".
  9. As result will be created a new ODS user and the link between ODS user and Facebook user will be established.
