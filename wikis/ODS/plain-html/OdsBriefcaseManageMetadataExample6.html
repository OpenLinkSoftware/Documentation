Gallery Photos Example for Extracting Metadata and DET based resources generation

Here is sample scenario:

  1. Go to http://<cname>/ods
  2. Log in as user
  3. Go to Gallery



  4. Upload a photo, for ex. b1.png



  5. Go to Briefcase path: DAV/home/user-name/Items/Graphics/



  6. View the content of the shown folders, for ex. view the "Dimensions" folder content:



  7. As result will be shown in the list a folder with name "800x516", where 800x516 are the dimensions of the uploaded from above image.
  8. Now go to DAV/home/user-name/Gallery



    • Note the folder "Gallery" will be created once your user creates Gallery instance.
  9. As result will be shown list of you Gallery albums
  10. Click on an album link



  11. As result will be shown list of the images contained in the selected album.
