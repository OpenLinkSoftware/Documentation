ODS Weblog Automatic Hyperlinking

  • If automatic hyperlinking is not desirable then whole post will not be annotated by putting only once <no-auto-href /> in the post body.
  • The tag currently recognized by phrase-match api for no links is <xmp> and we need the phrase api to implement more tags as PRE & CODE .
  • For multiple matches, only the first should be matched and turned to href link.
