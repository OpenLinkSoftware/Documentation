Weblog OAuth Ubiquity Authentication Example

The following example demonstrates oauth authentication getting the OAuth SID value for ex. for existing Weblog instance "myblog" with owner user demo at the OpenLink? Demo server.

In order to execute correctly the example, you need to have OAuth Generate Key for the Weblog "myblog" instance at OpenLink? Demo server. More information and simple scenario how to be done you can find here.

  1. Execute the command: 

    ods-oauth-host http://demo.openlinksw.com/OAuth

  2. Execute the command: 

    ods-set-mode oauth

  3. Setup OAuth key for your ODS application instance. Detail information you can find here.
  4. Go to http://demo.openlinksw.com/ods/oauth_sid.vsp
  5. In the shown form:
    1. Enter for user demo
    2. Choose the application "myblog".
    3. Click the "Execute" button.
  6. Enter for user demo its password and click the "Login" button.
  7. Click the "Authorize" button.
  8. Copy the obtains OAuth SID value for ex.: 6417767a4aa92635e7b57446e95ddc0d
  9. Execute the following command: 

    ods-set-weblog-oauth 6417767a4aa92635e7b57446e95ddc0d
