Feed Manager Example for Extracting Metadata and DET based resources generation

Here is sample scenario:

  1. Go to http://<cname>/ods
  2. Log in as user
  3. Go to Feed Manager -> Administration and subscribe for feed, for ex. http://www.openlinksw.com/blog/~kidehen/gems/rss.xml:



  4. As result you will be subscribed for the feed in question



  5. Go to Briefcase path: DAV/home/user-name/Items/Syndicated Content



  6. View the content of the shown folders, for ex. view the "title" folder content:



  7. As result will be shown in the list a folder with name "Kingsley Idehen's Blog Data Space", which is actually the name of the blog with the given RSS Feed file from above.
  8. Now go to DAV/home/user-name/Feed Subscriptions



    • Note the folder "Feed Subscriptions" will be created once your user creates Feed Manager instance.
  9. As result will be shown list of you Feed Manager subscribed for channels.
  10. Click on the channels link



  11. As result will be shown list of the subscription feeds.
