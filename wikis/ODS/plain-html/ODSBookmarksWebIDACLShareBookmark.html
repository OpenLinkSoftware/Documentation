ODS-Bookmark Manager WebID? Protocol Share Bookmark

  1. Install ODS-Bookmark Manager VAD package.



  2. Go to http://cname:port/ods and register user, for ex. with name demo and password demo.
    • Note: you can also use an existing ODS user for the next steps



  3. Logged in as user demo, click "Bookmarks" from the left vertical navigation.



  4. Click the "New Bookmark" icon from the main ODS-Bookmarks horizontal navigation or use the "Import" feature in order to add bookmarks(s) .







  5. Suppose a new Bookmark with name "OpenLink? Virtuoso Web Site" is created.
  6. The bookmark should be shown in the list of bookmarks for user demo.



  7. Click the "edit" link for the bookmark "OpenLink? Virtuoso Web Site".
  8. The Main properties tab for the bookmark should be shown.



  9. Go to the "Sharing" tab:



    • To share the bookmark to a person, see details here.
    • To share the bookmark to group of users, see details here.
    • To share the bookmark to public, see details here.
