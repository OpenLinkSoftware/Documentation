Importing contacts in Addressbook

Importing contacts using the Addressbook UI

You can import your contacts in ODS Addressbook using the Import feature by:

  • different type:
    1. File
    2. WebDAV?
    3. URL
    4. LDAP Server
  • content type:
    1. vCard
    2. CSV
    3. FOAF

Here is sample scenario of importing a vCard file kidehen.vcf:

  1. Go to http://demo.openlinksw.com/ods
  2. Log in as demo with password demo.
  3. Go to Addressbook.
  4. Click the "Import" button .
  5. Click the "Next" button.
  6. Leave "Content Type" to vCard and click the "Next" button.
  7. Click the "Browse" button and select a vCard file for ex. the kidehen.vcf file from your local file system.
  8. Click the "Next" button.
  9. In the shown form:
    1. Enter tags, for ex.: OpenLink?
    2. Check Validation fields if needed in order to keep the fields value in case there is already such contact in your Addressbook.
    3. Click the "Import" button.
  10. As result the contact(s) will be imported and shown in "My contacts" list.
