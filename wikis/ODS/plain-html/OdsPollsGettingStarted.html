ODS Polls Getting Started

Prerequisites

  1. Virtuoso Server has been installed and is running
  2. ODS and other ODS Applications including Polls have been successfully installed
  3. Register a new ODS account or login using your Virtuoso installers 'dba' or 'dav' accounts.

Step By Guide

  1. Using a web browser, login into you ODS user account or login as DAV or DBA by typing the following URL into your web browser: http://localhost:8890/ods/
  2. Select the Polls tab from the ODS Bar.
  3. Select the "Create New" Link.
  4. Enter your Polls Manager name, URL and click the button labeled "Create and Launch Application", which will create a new Polls instance.

Create Polls

Once you have created an instance, you are ready to create Polls, edit, delete,vote and view results:

  1. Click New Poll from the Polls Main Navigation Bar.
  2. Select poll type: single question or multiple questions. For example, select type: single question.
  3. In the offered form:
    1. Enter you question
    2. Enter a short description.
    3. Add tag(s)
    4. Choose Results visibility options.
    5. Choose Poll starting date / or manually to be started
    6. Choose Poll clsoing date / or manually to be closed.
  4. Click Next
  5. In the offered form::
    1. Select type pf multiple choice or number range
    2. Entered the expcted answers values
  6. Next you can "Preview" your Poll or to click Save in order to skip Preview.

Activate Polls

Once you have created a poll, in order to enable it you should:

  1. Click "My Polls" icon from the Polls Main Navigation Bar
  2. For a created Poll, shown in the list of the 2nd pane, click the link "Activate".
  3. Your poll is ready for user to vote to it.

Polls Page Overview

Check out some of Polls Managers other features as show below:

  1. Use the Navigation bar to view, create, delete and tag your polls.
  2. View polls by tag cloud shown in the 1st pane.
  3. View polls results by clicking the link "view" for a poll, shown in the 2nd pane
  4. Syndicate your polls in RSS, Atom, RDF, FOAF, SIOC(RDF/XML), SIOC(RDF/Turtle) formats
