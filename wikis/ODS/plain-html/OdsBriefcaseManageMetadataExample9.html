Bookmarks Example for Extracting Metadata and DET based resources generation

Here is sample scenario:

  1. Go to http://<cname>/ods
  2. Log in as user
  3. Go to Bookmarks and Import a file for ex. "my_bookmarks.xbel":



  4. As result its content will be imported and shown in the Bookmarks list:



  5. Go to Briefcase path: DAV/home/user-name/Items/Bookmarks (XBEL)



  6. View the content of the shown folders, for ex. view the "folderTitle" folder content:



  7. As result will be shown in the list all Bookmarks folders with their names presented as links.
  8. Now go to DAV/home/user-name/Bookmarks



    • Note the folder "Bookmarks" will be created once your user creates Bookmarks instance.
  9. As result will be shown list with your user's Bookmarks instances properties: bookmark, date, smart and tags.
  10. Click on the "bookmark" link



  11. As result will be shown list of the your user's Bookmarks instances.
