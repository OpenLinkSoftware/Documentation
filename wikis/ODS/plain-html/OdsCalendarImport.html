Importing events and tasks in ODS Calendar

In ODS Calendar you can import your tasks and events using the Import feature from :

  • File
  • WebDAV?
  • URL

Here is a sample scenario:

  1. Go to http://demo.openlinksw.com/ods
  2. Log in as demo with password demo
  3. Go to Calendar.
  4. Go to Import/Export from the left vertical navigation and choose Import.
  5. In the shown form:
    1. Select source type: File
    2. For Source name click the "Browse" button and select a .ics file form your local file system.
    3. Enter for Tags for ex.: ldp
    4. Click the "Import" button.
  6. As result your iCalendar file content will be imported.
