Wiki Example for Extracting Metadata and DET based resources generation

Here is sample scenario:

  1. Go to http://<cname>/ods
  2. Log in as user
  3. Go to Wiki and create article for ex. "MyTest?":



  4. Enter some body text and click "Save and release lock"



  5. Go to Briefcase path: DAV/home/user-name/Items/Wiki (OpenLink?)



  6. View the content of the shown folders, for ex. view the "Title" folder content:



  7. As result will be shown in the list a folder with name "MyTest?", which is actually the name of article created from above.
  8. Now go to DAV/home/user-name/wiki



    • Note the folder "wiki" will be created once your user creates wiki instance.
  9. As result will be shown list of you Wiki instances
  10. Click on an wiki instance link



  11. As result will be shown list of the current articles for the wiki instance and also will be shown:
    • "attach" folder: contains information for the attached files
    • "VVC" folder: contains the different versions of the wiki articles.
