Polls Manager

ODS-Polls Manager offers features to create, edit, or vote in a poll.

Feature Highlights

  • Customizable Polling Options
    • Create single- or multiple-answer polls.
    • Share results, or keep them private.
  • Option to add new questions to multiple-answer polls
  • Preview Poll
  • Tag polls
  • Multi-voting setting
  • Search within own and Public Polls
  • Automatic generation of syndication gems (Atom, RSS 2.0, RDF, FOAF, SIOC (RDF/XML), SIOC (N3/Turtle)
  • View settings:
    • Polls name display length limit.
    • Toolbar labels visibility.
    • Rows per page shown.
    • Atom File version choice.
  • Tag-cloud view
