Downstreaming (Importing) in Weblog

Downstreaming in Weblog can be performed from Settings->Downstream(Import).

Sample Scenario

Here is sample scenario:

  1. Install ODS and Weblog packages.
  2. Login at http://host:port/ods
  3. Go to Weblog from the ODS main vertical left navigation.
  4. Go to Settings->Downstream(Import).
  5. Enter for URL an rss for ex. http://www.openlinksw.com/blog/~kidehen/gems/rss.xml
  6. Click the "Next" button.
  7. Confirm with "Yes"
  8. As result will be shown message with number of imported posts.
  9. Go to "Home" in order to view the imported posts.
