Generate an X.509 Certificate hosted WebID? using IE browser

The following Guide provides the steps through the process of generating and exporting an X.509 hosted WebID? using IE: Note: these steps apply to any ODS instance where you have the requisite privileges.

  1. Set up an X.509 certificate issuer and HTTPS listener?
  2. Set up X.509 Certificate Service
  3. Go to the ODS home page (local or remote) and register new user



  4. Post registration you end up with an ODS generated Person Entity Identifier (an HTTP URI that Identifies "You") that takes the form: http://cname/dataspace/person/user-name#this



  5. Edit your profile (Note: If this step is skipped, the resulting X.509 certificate will not have a human readable name)



  6. Go to Security -> X.509 Certificates



  7. Click the "Generate" link.
    • Note: in other browser it will be shown as button.
  8. A notification should be shown:



  9. When finished, the OpenLink? X.509 Certificate Wizard will be shown:



  10. Click "Next"
  11. Select the desired certificate to be generated from the Wizard



  12. Click "Next"
  13. Enter the information for your certificate in the shown form



  14. Click "Next"
  15. Select the certificate's issuer, if appropriate:



  16. Click "Next"
  17. Specify what you would like to do with the generated certificate:



  18. Click "Next"
  19. As result will be shown the generated certificate.



  20. Click "Copy"
  21. Click "Exit"
  22. Back in the ODS Security -> X.509 Certificates, paste the copied file from above in the "X.509 Certificate" area:



  23. Hatch the "automatic login option" check-box



  24. Click the "Save Certificate" button which writes the Public Key Component of the generated Certificate to your FOAF based ODS hosted Profile data space.



Note: these steps apply to any ODS instance.
