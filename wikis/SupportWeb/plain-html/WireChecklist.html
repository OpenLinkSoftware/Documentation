Wire Protocol Drivers

OpenLink's provides wire protocol drivers and agents for MS SQL Server, MySQL, PostgreSQL, and Sybase.

OpenLink's Single-Tier (Lite) drivers are clientside-only drivers that establish their connection to the aforementioned database without the need for additional DBMS native client software. These drivers may reside on the DBMS server or a distinct client machine.

OpenLink's Multi-Tier database agents also establish connectivity to these DBMSs without the need for DBMS native client software or installation on the DBMS server.
