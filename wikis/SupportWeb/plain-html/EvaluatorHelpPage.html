Evaluator Start Page

UDA Driver Evaluators

  • All of our Data Access Drivers can be downloaded from the UDA site. We also provide a Driver Availability Wizard to help you find the options which suit your deployment environment. The Wizard cannot present every possible deployment configuration, so if you don't find something suitable, or can't decide between the options -- please contact us for assistance.
  • Our Installation Guides provide extensive installation instructions for all UDA products. Each Installation Guide also provides testing information and links to product pre-requisites docs.
  • The UDA Wiki Web covers advanced configuration instructions and native database configuration.
  • The UDA Wiki Web also provides links to troubleshooting resources and interpretation of diagnostic resources. Basic error message information can also be found in the general Technical FAQs and Mac FAQ.
  • Questions pertaining to product capabilities may be addressed using our UDA Driver Comparison Guide.
  • If you need additional assistance, feel free to open an online support case or visit our Discussion Forums.

Virtuoso Evaluators

  • The Virtuoso Universal Server and Client Connectivity Kit installers can be downloaded from the Virtuoso Download Page.
  • Our Installation Guides provide extensive installation instructions for the Universal Server and Client Connectivity Kit. Each Installation Guide also provides testing information and links to product pre-requisites docs.
  • The Virtuoso Wiki Web covers a wide range of topics pertaining to Virtuoso configuration and usage.
  • The OpenLink Data Spaces (ODS) and OpenLink AJAX Toolkit (OAT) Wiki Webs provide additional configuration and usage information for evaluators interested in the ODS application or OAT development kit.
  • Our Discussion Forums provides multiple forums for evaluators with questions pertaining to Virtual & Native Data Management, Web Services, OpenLink Data Spaces, and the AJAX Toolkit (OAT).
  • If you need additional assistance, feel free to open an online support case.
