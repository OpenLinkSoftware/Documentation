Apply License Files to Release 6.x Multi-Tier Data Access on Windows

  1. Retrieve your new OpenLink license file (e.g., sql_lt.lic, oplrqb.lic) from email or ODS briefcase.
  2. Place the new license file in the appropriate directory.
    • By default this is C:\Program Files\OpenLink Software\UDA\bin, but you may have installed elsewhere.
    • If in doubt, check all System and User Environment Variables ( System control panel >> Advanced tab >> Environment Variables) for an OPL_LICENSE_DIR.
    • If a license file by the same name is already present in this directory, rename the old file and save it until you're sure the new one works.
  3. Launch the Services or Component Services utility ( Start menu >> Control Panel >> Administrative Tools).
  4. Stop the OpenLink Request Broker service.
  5. Stop and restart the OpenLink License Manager service.
  6. Start the OpenLink Request Broker service.
