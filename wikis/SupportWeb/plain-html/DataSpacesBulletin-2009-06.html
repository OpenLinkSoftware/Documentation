MusicBrainz EC2 EMI Released

Musicbrainz joins, DBpedia, and Bio2Rdf in EC2 cloud.

MusicBrainz is a comprehensive, community-oriented metadatabase. It contains information about artists, album titles, track titles, track lengths, and so forth. You can access information pertaining to music by browsing the MusicBrainz site, or you can access it using various client applications. Better still, you can instantiate your own MusicBrainz instance in the cloud!

OpenLink Data Explorer Extension for Firefox, v0.25

ODEE is a browser-hosted client that uses REST-based "RDFizer" Proxy Services to produce RDF Linked Data from Web information resources. By default, ODEE is bound to a Virtuoso Sponger service provided and hosted by OpenLink Software. The Virtuoso Sponger is similar to an ODBC or JDBC Driver Manager, in that it connects client applications to one or more RDF Data Providers (also known as Sponger Cartridges). More than 30 such Cartridges are active on the OpenLink-hosted service, offering live, RDFized-on-the-fly versions of pages from Amazon, Bugzilla, eBay, Flickr, MusicBrainz, Yahoo! Finance, Twitter, and others, with still more on the way.

DBpedia ++

Did you know that DBpedia is powered by Virtuoso 5.0? This joint project offered up by OpenLink Software and two German universities has until recently served as the most notable front-end interface to a federated, Web-based OODBMS. OpenLink Software has now launched a veritable DBpedia++. LOD is billed as the bigger brother to DBpedia. LOD contains DBpedia, Freebase, MusicBrainz, Uniprot, Bio2Rdf, dbtune, NeuroCommons, PingTheSemanticWeb, and more. Add Yago, UMBEL, OpenCyc and reasoning to the LOD equation, and th quality of your search results and data meshes increases exponentially.

WebID?

OpenLink Virtuoso now provides a solution to trust and privacy issues surrounding Linked Data. Virtuoso's new WebID? support uses SPARQL- based modifications to SSL to provide Identity-based, platform independent security. WebID? solves innumerable headaches such as centralized certificate management, and it promotes the development of such things as highly granular, policy based security. Learn how WebID? can identity challenges can stop the nuisance of spam, malicious commentary, and much, much more.

Entity Ranking

Entity Ranking joins Faceted Browsing as a Virtuoso 6.0 deliverable. Entity Ranking is a visual aid that promotes identification of highly- relevant search results. It is the Linked Data Web equivalent of Page Rank, but it is all the more powerful, because it uses a link coefficient algorithm fueled by the relationships between the entities in the linked data search.

ODS-Briefcases & Amazon S3 Buckets

OpenLink has realized the final piece of its Unified Storage story. Amazon S3 buckets now join WebDav and SQL data as resources transparently integrated into -- and exposed by -- OpenLink ODS-Briefcases. S3 buckets, WebDav, and SQL are all now accessible using any of several different views: OS filesystem tree, RDF graph, or relational table tuples.
