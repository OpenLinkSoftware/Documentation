Apply License Files for Virtuoso Release 4.x and earlier, on Mac OS X

  1. Open your Finder.
  2. Click the Applications icon.
  3. Open the Virtuoso folder.
  4. Double Click the stop script that pertains to your Virtuoso server instance.
  5. Retrieve your new OpenLink license file from an email or ODS briefcase.
  6. Overwrite the old license with the new one. It is likely to appear in /Library/Application Support/openlink/bin.
  7. Go back to the Virtuoso folder.
  8. Double Click the start script that pertains to your Virtuoso server instance.
