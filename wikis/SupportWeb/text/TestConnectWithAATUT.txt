---++Create ODBC DSNs with the Multi-Tier Admin Assistant (All Platforms)

This presentation teaches you how to create ODBC DSNs using the Multi-Tier Admin Assistant. The Admin Assistant is a Web-GUI that ships with the Multi-Tier server components. The DSNs that it creates are local to the machine that hosts the <nop>OpenLink Request Broker, a Multi-Tier server component. (Time: 7:58)
