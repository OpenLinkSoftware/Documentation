---++Shutdown the <nop>OpenLink License Manager

1. Expand the Start menu.

2. Click the Control Panel.

3. Open the Administrative Tools sub-folder, if it exists.

4. Launch the Services utility.

5. Shutdown the <nop>OpenLink Request Broker.

6. Shutdown the <nop>OpenLink License Manager service.
