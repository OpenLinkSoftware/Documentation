---++ Legacy Support & Deprecation

%NAVY%This section addresses <nop>OpenLink's legacy support and product deprecation policies.%ENDCOLOR%

<nop>OpenLink Software stands behind each of the products in its product portfolio. The <nop>OpenLink Technical Services team will provide Web-based support for all of <nop>OpenLink's versions and releases. However, the team will prompt you to upgrade if compatibility, enhancement needs, or other issues are encountered in a product whose code base has been closed. 

The current release of <nop>OpenLink Data Access Drivers is 6.2. <nop>OpenLink's 1x-5.x Data Access Driver code bases are closed and the associated products are considered deprecated. Upgrade to 6.x will be necessary for product enhancements and compatibility issues, unless fee-based custom portation is arranged.

The current release of <nop>OpenLink Virtuoso Universal Server is 5.x. <nop>OpenLink's previous Virtuoso code bases are closed and considered deprecated. Upgrade to 5.x will be necessary for product enhancements and compatibility issues, unless fee-based custom portation is arranged.
