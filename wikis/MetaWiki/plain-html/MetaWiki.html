OpenLink Wiki Server

Welcome to the OpenLink Software wiki server.

Here you can find documentation on our Universal Data Access (UDA) driver suite for ODBC, JDBC, OleDB and ADO.NET; for the Virtuoso SQL and RDF database; for the OpenLink Dataspace Application Suite; and for the OpenLink AJAX Toolkit (OAT).

Each of these projects has authoritative documentation by OpenLink? staff, implemented as a wiki cluster:

  • Universal Data-Access Suite (UDA)
  • Virtuoso Universal Server
  • OpenLink Data Spaces (ODS)
  • OpenLink AJAX Toolkit (OAT)
