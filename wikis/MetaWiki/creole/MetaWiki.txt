  OpenLink Wiki Server 
**Welcome to the OpenLink Software wiki server. ** 

Here you can find documentation on our Universal Data Access ( [[http://uda.openlinksw.com/|UDA]] ) driver suite for ODBC, JDBC, OleDB and ADO.NET; for the [[http://virtuoso.openlinksw.com|Virtuoso]] SQL and RDF database; for the [[http://virtuoso.openlinksw.com/wiki/main/Main/Ods|OpenLink Dataspace]] Application Suite; and for the OpenLink AJAX Toolkit ( [[http://oat.openlinksw.com/|OAT]] ). 

Each of these projects has authoritative documentation by [[OpenLink|OpenLink]] staff, implemented as a wiki cluster: 

* [[http://wikis.openlinksw.com/dataspace/owiki/wiki/UdaWikiWeb/|Universal Data-Access Suite (UDA)]]   
* [[http://wikis.openlinksw.com/dataspace/owiki/wiki/VirtuosoWikiWeb/WelcomeVisitors|Virtuoso Universal Server]]   
* [[http://wikis.openlinksw.com/dataspace/owiki/wiki/OdsWikiWeb/|OpenLink Data Spaces (ODS)]]   
* [[http://wikis.openlinksw.com/dataspace/owiki/wiki/OATWikiWeb/|OpenLink AJAX Toolkit (OAT)]]
