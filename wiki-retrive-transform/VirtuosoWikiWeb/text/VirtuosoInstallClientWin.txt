== Installation ==
# The Client Connectivity Suite installer ships as a .msi file: 
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin01.png" style="wikiautogen"/>%BR%%BR%
# Double click on the installer icon to commence the installation:
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin02.png" style="wikiautogen"/>%BR%%BR%
# Read and accept the license agreement:
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin03.png" style="wikiautogen"/>%BR%%BR%
# Select the type of installation to perform -- Typical, Complete, or Custom.  
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin04.png" style="wikiautogen"/>%BR%%BR%
# Typical and Complete installations go to the default location, {{{C:\Program Files\OpenLink Software\Virtuoso\}}}. For Custom installation, you may choose the installation directory or accept the default:
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin05.png" style="wikiautogen"/>%BR%%BR%
# Typical and Complete installations include drivers and providers for all supported client mechanisms.  For Custom installation, you may specify a more limited selection:
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin06.png" style="wikiautogen"/>%BR%%BR%
# Click the install button to commence the installation:
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin07.png" style="wikiautogen"/>%BR%%BR%
# The installation progress is displayed:
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin08.png" style="wikiautogen"/>%BR%%BR%
# The installation is now complete.
%BR%%BR%<img src="%ATTACHURLPATH%/VirtuosoInstallClientWin09.png" style="wikiautogen"/>%BR%%BR%
