Sharing in ODS Addressbook

ODS Addressbook offers the feature for sharing contact(s) to other user in the relevant ODS dataspace. Here is a sample scenario:

  1. Go to your Addressbook instance
  2. From the 2nd pane, under My contact, select the desired contact(s) to be shared to.
  3. Click the "Sharing" icon from the Main Addressbook Horizontal navigation bar.
  4. As result will be opened the the Sharing to user(s) form.
  5. Click the "Select" button to choose ODS user(s) or enter then directly in the "User(s)" field.
  6. For ex. if you click the "Select" button, will be opened new window with the list of the ODS registered users. Select directly user(s) or make filter. In our example we will select the test1 user.
  7. You can leave the check-box "Override current sharings" checked:
  8. Click the "Share" button.
  9. Now let's check for user test1 what does it have in its Addressbook instance. Log in as user test1
  10. Go to Addressbook
  11. As result will be opened the test1's Addressbook instance.
  12. Click the "Shared Contacts" icon from the Main Addressbook Horizontal navigation bar.
  13. In the "Shared Contacts" list expand the "Shared Contacts By" node.
  14. As result for every user that had shared to user test1 resources, will be shown separate folder labeled with the users name.
  15. Expand the "demo" user folder
  16. As result in the 2nd pane will be shown the shared contacts from user demo to the logged in user test1. In our example this is the contact for Kingsley Idehen.
