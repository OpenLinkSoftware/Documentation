Polls Controllers

Poll get

  • Description: gets poll by given id
  • API name: ODS.ODS_API."poll.get"
  • Parameters:
    • poll_id

Poll new

  • Description: creates new poll
  • API name: ODS.ODS_API."poll.new"
  • Parameters:
    • inst_id
    • name
    • description
    • tags
    • multi_vote
    • vote_result
    • vote_result_before
    • vote_result_opened
    • date_start
    • date_end
    • mode

  • Description: creates new question to existing poll
  • API name: ODS.ODS_API."poll.question.new"

    • text

    • required
    • type
    • answer
