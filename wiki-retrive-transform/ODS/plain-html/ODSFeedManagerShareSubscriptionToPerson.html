ODS-Feed Manager WebID? Protocol Share Subscription to Person

  1. Make sure the initial steps from the ODS Feed Manager WebID Protocol Share Subscription Guide are performed.
  2. Set Personal WebID? ACL Inheritance to be applied automatically to every new subscription you add in your Feed Manager or
  3. Manually set the Personal WebID? ACL by going for a given subscription in ODS-Feed Manager -> Administration -> Feeds -> Subscription -> Edit -> Sharing tab.
  4. In the "WebID? users" section click the "Add" button in order to add new WebID? and set respectively the desired rights.



  5. In the shown form:
    1. Select for "Access Type": Personal
    2. Enter for User's WebID? an user's WebID? for which user is created WebID certificate for ex. http://localhost:8896/dataspace/person/john#this
    3. Set Web Access, for example by checking the (R)ead check-box.



    4. Click the "Save" button.
