Importing in Bookmarks

You can import your bookmarks using the Import feature of the ODS Bookmarks by:

  • Source type:
    1. File
    2. WebDAV?
    3. URI
    4. del.icio.us

Here is a sample scenario:

  1. Go to http//demo.openlinksw.com/ods
  2. Login as demo with password demo
  3. Go to Bookmarks
  4. Click the "Import" button from the Main Horizontal Bookmarks Navigation.
  5. Click the "Next" button.
  6. Click the "Browse" button and select a bookmark file from your local file system.
  7. Click the "Next" button.
  8. In the shown form:
    1. You can enter a desired folder the bookmarks to be imported too or leave this field empty. For ex. you can value "test"
    2. You can add tag(s) for ex.: new or to chose from the shown list of "My Tags".
  9. Click the "Import" button.
  10. As result will be shown the window with progress bar. Here you can:
    1. Leave the process until the import finishes to 100%.
    2. Click the "Background" button, which will cause the process to run in background mode.
    3. Click the "Stop" button which will stop the import process.
  11. As result the bookmarks will be imported.
