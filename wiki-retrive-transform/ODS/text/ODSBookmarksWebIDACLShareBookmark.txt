---+ ODS-Bookmark Manager WebID Protocol Share Bookmark

   1 Install [[https://virtuoso.openlinksw.com/download/][ODS-Bookmark Manager]] VAD package.
%BR%%BR%<img src="%ATTACHURLPATH%/bc10.png" />%BR%%BR%
   1 Go to http://cname:port/ods and register user, for ex. with name demo and password demo.
      * Note: you can also use an existing ODS user for the next steps
%BR%%BR%<img src="%ATTACHURLPATH%/ac11.png" />%BR%%BR%
   1 Logged in as user demo, click "Bookmarks" from the left vertical navigation.
%BR%%BR%<img src="%ATTACHURLPATH%/bc12.png" />%BR%%BR%
   1 Click the "New Bookmark" icon from the main ODS-Bookmarks horizontal navigation or use the "Import" feature in order to add bookmarks(s) .
%BR%%BR%<img src="%ATTACHURLPATH%/bc13.png" />%BR%%BR%
%BR%%BR%<img src="%ATTACHURLPATH%/bc14.png" />%BR%%BR%
   1 Suppose a new Bookmark with name "OpenLink Virtuoso Web Site" is created.
   1 The bookmark should be shown in the list of bookmarks for user demo.
%BR%%BR%<img src="%ATTACHURLPATH%/bc15.png" />%BR%%BR%
   1 Click the "edit" link for the bookmark "OpenLink Virtuoso Web Site".
   1 The Main properties tab for the bookmark should be shown.
%BR%%BR%<img src="%ATTACHURLPATH%/bc16.png" />%BR%%BR%
   1 Go to the "Sharing" tab:
%BR%%BR%<img src="%ATTACHURLPATH%/bc17.png" />%BR%%BR%
      * To share the bookmark to a person, see details [[ODSBookmarksShareBookmarkToPerson][here]].
      * To share the bookmark to group of users, see details [[ODSBookmarksShareBookmarkToGroup][here]].
      * To share the bookmark to public, see details [[ODSBookmarksShareBookmarkToPublic][here]].
