---+ ODS-Calendar WebID Protocol Share Task

   1 Install [[https://virtuoso.openlinksw.com/download/][ODS-Calendar]] VAD package.
%BR%%BR%<img src="%ATTACHURLPATH%/cc10.png" />%BR%%BR%
   1 Go to http://cname:port/ods and register user, for ex. with name demo and password demo.
      * Note: you can also use an existing ODS user for the next steps
%BR%%BR%<img src="%ATTACHURLPATH%/ac11.png" />%BR%%BR%
   1 Logged in as user demo, click the link "Calendar" from the left vertical navigation.
%BR%%BR%<img src="%ATTACHURLPATH%/cc12.png" />%BR%%BR%
   1 Click the "New Task" icon from the main ODS-Calendar horizontal navigation or use the "Import" feature in order to add task(s) .
%BR%%BR%<img src="%ATTACHURLPATH%/cc13.png" />%BR%%BR%
%BR%%BR%<img src="%ATTACHURLPATH%/cc14.png" />%BR%%BR%
   1 Suppose a new Task with name "mytest" is created.
   1 The task should be shown in the list of Tasks for user demo.
%BR%%BR%<img src="%ATTACHURLPATH%/cc15.png" />%BR%%BR%
   1 Click the "edit" link for the created task.
   1 The Main properties tab for the task should be shown.
%BR%%BR%<img src="%ATTACHURLPATH%/cc16.png" />%BR%%BR%
   1 Set "Privacy" to "ACL":
%BR%%BR%<img src="%ATTACHURLPATH%/cc16a.png" />%BR%%BR%
   1 As result the "Sharing" tab will be shown. Go to it:
%BR%%BR%<img src="%ATTACHURLPATH%/cc17.png" />%BR%%BR%
      * To share the task to a person, see details [[ODSCalendarShareTaskToPerson][here]].
      * To share the task to group of users, see details [[ODSCalendarShareTaskToGroup][here]].
      * To share the task to public, see details [[ODSCalendarShareTaskToPublic][here]].
