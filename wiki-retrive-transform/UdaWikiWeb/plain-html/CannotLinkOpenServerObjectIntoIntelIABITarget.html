Cannot link OpenServer object into Intel iABI target

"Cannot link OpenServer object into Intel iABI target" errors are serious. They occur when your Progress database and your OpenLink Progress Database Agent are linked against different Application Binary Interfaces. Contact OpenLink Product Support, to determine if new agents have been built to accommodate this version of Progress.

Evidence

uname -a
Database Agent Version String
DBMS Version
