Relink Scripts

The <nopOpenLink Database Agent Relink Scripts are used to recompile Multi-Tier database agents on Unix or Linux platforms. They are found in the /lib sub-directory of your server's OpenLink installation. The filenames begin with ld, and end with an abbreviated code which represents the database agent to be relinked. For example:

ldora8 - Oracle 8 Relink Script

ldpgr95 - Postgres '95 Relink Script

Progress users will also have an mkdlc8 or mkdlc9 script. Please submit copies of your relink script(s) to OpenLink Product Support.
