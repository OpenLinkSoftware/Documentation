Binary Directory

The Binary Directory parameter (BinaryDirectory) appears in the [Request Broker] section of your OpenLink Session Rules Book. This parameter must be set to the /bin sub-directory of your OpenLink installation. The Broker uses the location passed by this parameter to locate the database agents and Session Rules Books. The Broker will also create log files in this directory, by default.

Example: BinaryDirectory=/usr/openlink/bin
