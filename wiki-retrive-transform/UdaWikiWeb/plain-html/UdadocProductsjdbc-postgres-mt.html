The Multi-Tier JDBC driver for PostgreSQL? consists of a set of generic JDBC client classes and the PostgreSQL? Database agent on the server the built in database independent network layer for communication between the client and server.

The OpenLink? Multi-Tier Drivers for PostgreSQL? are a Type B class driver developed using the PostgreSQL? network (wire) protocol interface.

These drivers are currently available for PostgreSQL? versions 6.x, 7.x and beyond.
