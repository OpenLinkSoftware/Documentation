The Multi-Tier .Net Provider for Oracle consists of a 100% managed client assembly and the Oracle Database agent on the server utilizing the built in database independent network layer for communication between the client and server.

The OpenLink? Multi-Tier Drivers for Oracle are a Type A class driver developed using the OCI call level interface.

These drivers are currently available for Oracle versions 7.x, 8.x, 9.x and beyond.
