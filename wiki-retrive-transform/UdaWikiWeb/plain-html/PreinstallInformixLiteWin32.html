DBMS Requirements

You need to know a variety of information pertaining to your target Informix instance and database:

  • Database Name
  • Informix Instance Name
  • Informix Instance Startup Mode (Sockets or Shared Memory). Note: Shared Memory connections with the Lite Edition are only possible when it is running on the Informix instance host. Sockets connections with the Lite Edition may be made from any host.
  • Informix Service Name
  • Informix TCP Port Number
  • Database Server Hostname or IP Address

Software Requirements

  • You need to know whether the client application is 32-bit or 64-bit. The Single-Tier "Lite" Edition ODBC Driver for Informix must match the bit format of the client application.
  • The Informix native client or Informix itself must be installed on the same host as the Single-Tier "Lite" Edition ODBC Driver for Informix. This software is available free of charge from the IBM web site, as part of the Informix Client SDK.
    • The local Informix components must match the bit format of the client application and the Single-Tier "Lite" Edition ODBC Driver for Informix.

Configuration Requirements

  • You need to configure your Informix client for connectivity to the remote Informix DBMS.
