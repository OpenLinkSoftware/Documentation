Host Name

The Host Name parameter appears in a variety of places, and it takes on a variety of forms. You will see variants of Host Name in the Administrative Assistant, odbc.ini file, udbc.ini file, ODBC Data Sources Administrator, and JDBC connection URL's. In every instance, the Host name field passes the name or IP address of the machine with the OpenLink Request Broker.

Be advised, if you hardcode the Host Name into the [generic_xxx] section of your Session Rules Book, it will override the Host Name values passed by your DSNs.

Variants: /HOST, Host, HostName, Hostname
