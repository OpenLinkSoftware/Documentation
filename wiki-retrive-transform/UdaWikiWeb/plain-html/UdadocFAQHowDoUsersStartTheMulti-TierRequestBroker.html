  • Login to the machine, which contains the OpenLink? server components.
  • Open the Services utility. (Control Panel)
  • Locate the OpenLink? Request Broker service.
  • Highlight the service.
  • Click the Start icon.
  • Login to the machine, which contains the OpenLink? server components.
  • cd into the root of the OpenLink? installation.
  • Execute openlink.sh (Bourne/Bash shell) or openlink.csh (C shell).
