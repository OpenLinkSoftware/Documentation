Lost RPC connection to heterogeneous remote agent

Do you encounter "ORA-28511: lost RPC connection to heterogeneous remote agent" errors when you attempt to connect Oracle's HSODBC to OpenLink drivers? This error is associated with inappropriate values in OpenLink Data Source Names. To resolve the problem, provide OpenLink with a copy of your client machine's odbc.ini file (unix) or the attributes that you supply in your Data Source Name configuration dialog (Windows/Mac).
