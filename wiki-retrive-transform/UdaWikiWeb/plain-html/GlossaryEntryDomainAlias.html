Domain Alias

Domain Aliases may be found in the [Domain Aliases] section of the OpenLink Session Rules Book. You will also find Domain Aliases in the Edit Domain Aliases portion of the Administrative Assistant. The standard Domain Alias is a shortened form of a value passed in a DSN's Domain Name or Server Type field. For instance, Oracle 8 pro90b

This allows you to create a single Mapping Rule, which applies in a variety of cases. This promotes economical and flexible security. If you do not employ regular expressions, you will need to create one Mapping Rule per each Domain.
