Installing Internet Plugin components for Safari

Safari extensions are not allowed to perform some functions. To deliver full features of the HTML5 ODBC Bridge to Safari users, some features must be installed as an OS extension.





  • For Mac OS X

  • For Windows



For Mac OS X

  1. Download the ODBC Internet Plugin installer for Mac OS X.
  2. Double click on the HTML5_WebDB2ODBC?_NPAPI_Plugin.dmg file.





  3. Double click on the HTML5 WebDB?-to-ODBC (Bridge) NPAPI Plugin.mpkg package.





  4. Click continue from the installer Welcome dialog to continue the installation.





  5. Click install to begin installation.





  6. Enter password to authorize installation.





  7. Installation is complete.




For Windows

  1. Download the ODBC Internet Plugin installer for Windows.
  2. Double click on the Setup_HTML5_WebDb2ODBC?_Plugin.exe installer to begin the installation.





  3. Select Next from the installer Welcome dialog to continue the installation





  4. Select the destination location for the installation and click Next





  5. Click the Install to install the components





  6. Installation is complete
