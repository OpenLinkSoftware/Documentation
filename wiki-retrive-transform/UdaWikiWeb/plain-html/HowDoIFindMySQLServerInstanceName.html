Identifying the Microsoft SQL Server Instance Name

Microsoft SQL Server 2000

  1. Log in to your Microsoft SQL Server database server.
  2. Launch the Microsoft SQL Server Network Utility (Start menu >> Programs >> Microsoft SQL Server).
  3. Microsoft SQL Server instance names should be immediately visible.

Microsoft SQL Server 2005

  1. Log in to your Microsoft SQL Server database server.
  2. Launch the Microsoft SQL Server 2005 Configuration Manager (Start menu >> Programs >> Microsoft SQL Server).
  3. Microsoft SQL Server instance names should be immediately visible.
