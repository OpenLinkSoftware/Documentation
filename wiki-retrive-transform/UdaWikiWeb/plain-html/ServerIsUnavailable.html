Error Message: Server is unavailable or does not exist (20009)

The most common causes of this error are --

  • an incorrect value for the target SQL Server (Microsoft or Sybase) hostname or IP address
  • an incorrect value for TCP/IP listen port for Microsoft SQL Server or Sybase
  • an incorrect value for the target Microsoft SQL Server Instance Name

If you are certain these are all correct, check Microsoft's Knowledgebase Article on this error message.

Come back to us if that doesn't provide the solution.
