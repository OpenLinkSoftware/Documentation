==Ifsame database==

Ifsame database is a database agent <nop>ReUse rule value. Ifsame database specifies that all requests for the same database name will be serviced by one agent. Be advised, database aliases have no bearing on this rule. Instead, the Broker polls for the physical database identifier passed in a DSN or serverside DATABASE parameter.
