===No key columns found for table referenced by keyset driven cursor===

"No key columns" errors arise when you run queries against tables, which do not have primary keys. To remedy this situation, define primary keys for your tables. Alternatively, select a different cursor modality.

Note: PHP users can recompile PHP with FORWARDONLY support by 0 default.
