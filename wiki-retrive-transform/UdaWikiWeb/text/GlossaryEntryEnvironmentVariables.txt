==Environment Variables==

An Environment Variables list box occurs in the Administrative Assistant's Database Agent Settings Form and Wizard. This box contains variables, which must be set for an agent to function. The Environment Variables box corresponds to the Environment parameters, which you may find in the <nop>OpenLink Session Rules Book's [generic_xxx] sections.
