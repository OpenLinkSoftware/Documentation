The ODBC .Net Data Providers enables connectivity to any ODBC Data Source by acting as a Bridge between ADO.Net and ODBC. This Provider offers ADO.NET access to existing ODBC Data Sources from an ADO.NET client environment. 

The [[OpenLink]]  ODBC .Net Data Providers are a Type C class of provider. 

 
 [[/images/aodbcst.gif|]]  

 
/ ODBC .Net Data Provider Architecture Diagram Image scaled down; Click to enlarge./ 

 
This driver is available for Windows and Linux Operating systems.
