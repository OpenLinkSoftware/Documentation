The Multi-Tier JDBC driver for Oracle consists of a set of generic JDBC client classes and the Oracle Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for Oracle are a Type A class driver developed using the OCI call level interface. 

These drivers are currently available for Oracle versions 7.x, 8.x, 9.x and beyond. 

 
 [[/images/aodbcst.gif|]]  

 
/Oracle Multi-Tier JDBC Driver Architecture Diagram Image scaled down; Click to enlarge./ 

 
This driver is available for Windows, Linux, Unix(most flavours), Mac Classic, Mac OS X or any Java enabled Operating systems.
