= Pre-Installation Requirements for the Single-Tier "Lite" Edition ODBC Driver for <nop>MySQL, for Windows =

%TOC%

== DBMS Requirements==

You need to know a variety of information pertaining to your target <nop>MySQL instance and database:

* [[HowDoIFindMyMySQLListeningPort|<nop>MySQL TCP port]]
* Hostname or IP address of the <nop>MySQL database server 

== Software Requirements ==

* You need to know whether the client application is 32-bit or 64-bit. The Single-Tier "Lite" Edition ODBC Driver for <nop>MySQL must match the bit format of the client application.

== Configuration Requirements ==

* There are no software configuration requirements associated with the Single-Tier "Lite" Edition ODBC Driver for <nop>MySQL.
