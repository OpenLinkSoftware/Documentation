==Row Buffer Size==

The Row Buffer Size parameter appears in the ODBC Datasources Administrator. The value passed by Row Buffer Size represents the number of rows of resultset data which you want fetched before data is returned to the client. A high Row Buffer Size reduces the need to traverse back and forth across the network. However, a high Row Buffer Size will result in longer intervals between the arrival of data.   

See Also: FBS, Fetch Buffer Size
