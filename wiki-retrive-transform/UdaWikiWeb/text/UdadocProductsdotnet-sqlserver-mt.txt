The Multi-Tier .Net Provider for MS SQL Server consists of a 100% managed client assembly and the MS SQL Server Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier providers for MS SQLServer are a Type B class driver developed using the TDS network (wire) protocol interface. 

These providers are currently available for MS SQLServer versions 6.5, 7, 2000 and beyond.
