= Error Message: {{{Unable to connect: SQLServer is unavailable or does not exist (20009)}}} =

The most common causes of this error are --

* an incorrect value for the target SQL Server (Microsoft or Sybase) hostname or IP address
* an incorrect value for TCP/IP listen port for [[HowDoIFindMySQLServerListenPort| Microsoft SQL Server]] or [[HowDoIFindMySybaseListenPort| Sybase]]
* an incorrect value for the target Microsoft SQL Server [[HowDoIFindMySQLServerInstanceName| Instance Name]]

If you are certain these are all correct, check [[http://support.microsoft.com/kb/328306| Microsoft's Knowledgebase Article]] on this error message.

[[http://support.openlinksw.com/support/online-support.vsp| Come back to us]] if that doesn't provide the solution.
