==Send Size==

Send Size is a widespread parameter which appears in each of <nop>OpenLink's configuration files and utilities. Send Size may be set on both the client and server. It represents the size of a component's RPC send buffer. When this buffer is full, outgoing packets will be sent across the network to the other side. Furthermore, your client and server Send Size values should be identical. However, you do not want to alter these parameters without guidance from <nop>OpenLink Product Support. 

See Also: Send Buffer, Receive Size
