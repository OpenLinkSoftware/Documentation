= Pre-Installation Requirements for the Multi-Tier "Enterprise" Edition Generic Client ODBC Drivers for Unix-like OS =

%TOC%

== DBMS Requirements ==

* There are no DBMS requirements associated with the Multi-Tier "Enterprise" Edition Generic Client ODBC Drivers.

== Software Requirements ==

* You need to know whether the client application is 32-bit or 64-bit. The Multi-Tier "Enterprise" Edition Generic Client ODBC Drivers must match the bit format of the client application.

== Configuration Requirements ==

* There are no software configuration requirements associated with the Multi-Tier "Enterprise" Edition Generic Client ODBC Drivers.
