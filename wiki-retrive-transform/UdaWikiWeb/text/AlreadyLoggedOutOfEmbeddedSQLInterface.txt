===Already Logged Out of Embedded SQL Interface (A)===

The "Already logged out of embedded SQL interface" error message sometimes occurs when "Lite" Edition users close one connection and attempt to open another. This is a known issue which is currently being addressed by Development.

In the interim, <nop>OpenLink suggests that users install the <nop>OpenLink Multi-Tier client and server components on their PC clients. This enables users to connect to backend databases via the database native client. Hence, the Multi-Tier drivers perform in the same manner as Lite.

===Evidence===

* Single-Tier Version, Date, and File Information
* ODBC Client Trace
