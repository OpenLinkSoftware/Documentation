==EMERG==

EMERG is a Request Broker log detail setting. It appears in the Administrative Assistant's Edit Request Broker's Parameters Form and Wizard. If you set your Broker's log detail to EMERG, the Broker will write only "system unusable" error messages to it's log file. 

This setting is not optimal for diagnostic purposes.
