==Send Buffer==

The Send Buffer parameter appears in the Administrative Assistant's Edit Request Broker's Parameters Form and Wizard. Send Buffer represents the size of the server's RPC send buffer. When this buffer is full, outgoing packets will be sent across the network to the client. Furthermore, your client Send Size and your server's Send Buffer values should be identical. However, you do not want to alter these parameters without guidance from <nop>OpenLink Product Support   

Variants: Send Size   

See Also: Send Size, Receive Size
