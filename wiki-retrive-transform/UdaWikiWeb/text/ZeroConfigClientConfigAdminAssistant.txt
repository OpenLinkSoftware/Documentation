= Configuring a Multi-Tier Client to use a <nop>ZeroConfig DSN through the HTTP-based Admin Assistant =

When configuring a DSN in the HTTP-based Admin Assistant, the Server field will offer a drop-down menu of available Zero Config targets --

During the configuration of a DSN in the Admin Assistant it is currently not possible to dynamically discover and present the Zero Config entries. However, you can manually put a Zero Config entry in the Server field.
