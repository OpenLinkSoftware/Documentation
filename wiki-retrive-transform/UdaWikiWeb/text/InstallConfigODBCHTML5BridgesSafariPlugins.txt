= Installing Internet Plugin components for Safari =

Safari extensions are not allowed to perform some functions.  To deliver full features of the [[InstallConfigODBCHTML5Bridge|HTML5 ODBC Bridge]] to Safari users, some features must be installed as an OS extension. 

%TOC%

== For Mac OS X ==

# Download the [[http://opldownload.s3.amazonaws.com/HTML5/ODBC/HTML5_WebDB2ODBC_NPAPI_Plugin.dmg| ODBC Internet Plugin]] installer for Mac OS X.
# Double click on the <b>HTML5_WebDB2ODBC_NPAPI_Plugin.dmg</b> file.
%BR%%BR%<img src="%ATTACHURLPATH%/SafariIODBCplugin01.png" style="wikiautogen"/>%BR%%BR%
# Double click on the **HTML5 WebDB-to-ODBC (Bridge) NPAPI Plugin.mpkg** package.
%BR%%BR%<img src="%ATTACHURLPATH%/SafariIODBCplugin02.png" style="wikiautogen"/>%BR%%BR%
# Click **continue** from the installer Welcome dialog to continue the installation.
%BR%%BR%<img src="%ATTACHURLPATH%/SafariIODBCplugin03.png" style="wikiautogen"/>%BR%%BR%
# Click **install** to begin installation.
%BR%%BR%<img src="%ATTACHURLPATH%/SafariIODBCplugin04.png" style="wikiautogen"/>%BR%%BR%
# Enter **password** to authorize installation.
%BR%%BR%<img src="%ATTACHURLPATH%/SafariIODBCplugin05.png" style="wikiautogen"/>%BR%%BR%
# Installation is complete.
%BR%%BR%<img src="%ATTACHURLPATH%/SafariIODBCplugin06.png" style="wikiautogen"/>%BR%%BR%

== For Windows ==

# Download the [[http://opldownload.s3.amazonaws.com/HTML5/ODBC/Setup_HTML5_WebDb2ODBC_Plugin.exe| ODBC Internet Plugin]] installer for Windows.
# Double click on the <b>Setup_HTML5_WebDb2ODBC_Plugin.exe</b> installer to begin the installation.
%BR%%BR%<img src="%ATTACHURLPATH%/SetupIODBCPlugin01.png" style="wikiautogen"/>%BR%%BR%
# Select **Next** from the installer Welcome dialog to continue the installation
%BR%%BR%<img src="%ATTACHURLPATH%/SetupIODBCPlugin02.png" style="wikiautogen"/>%BR%%BR%
# Select the **destination** location for the installation and click <code>Next</code>
%BR%%BR%<img src="%ATTACHURLPATH%/SetupIODBCPlugin03.png" style="wikiautogen"/>%BR%%BR%
# Click the **Install** to install the components
