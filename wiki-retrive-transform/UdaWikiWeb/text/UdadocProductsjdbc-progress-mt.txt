The Multi-Tier JDBC driver for Progress consists of a set of generic JDBC client classes and the Progress Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for Progress are a Type A class driver developed using the ESQL/C and SQL-92 call level interface. 

These drivers are currently available for Progress versions 7.x, 8.x, 9.x and beyond. 

 
 [[/images/aodbcst.gif|]]  

 
/Progress Multi-Tier JDBC Driver Architecture Diagram Image scaled down; Click to enlarge./ 

 
This driver is available for Windows, Linux, Unix(most flavours), Mac Classic, Mac OS X or any Java enabled Operating systems.
