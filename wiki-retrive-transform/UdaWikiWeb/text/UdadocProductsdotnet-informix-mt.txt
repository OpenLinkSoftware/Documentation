The Multi-Tier .Net Provider for Informix consists of a 100% managed client assembly and the Informix Database agent on the server utilizing the built in database independent network layer for communication between the client and server. 

The [[OpenLink]]  Multi-Tier Drivers for Informix are a Type A class driver developed using the ESQL/C call level interface. 

These drivers are currently available for Informix versions 7.x, 9.x(IDS 2000) and beyond.
