---+ Evaluator Start Page

---++ UDA Driver Evaluators

   * All of our Data Access Drivers can be downloaded from the [[http://uda.openlinksw.com/][UDA site]].  We also provide a [[http://download.openlinksw.com/solwiz/][Driver Availability Wizard]] to help you find the options which suit your deployment environment.  The Wizard cannot present every possible deployment configuration, so if you don't find something suitable, or can't decide between the options -- please [[http://www.openlinksw.com/contact][contact us for assistance]].

   * Our [[Installation Guides]] provide extensive installation instructions for all UDA products. Each Installation Guide also provides testing information and links to product pre-requisites docs.

   * The UDA Wiki Web covers [[http://wikis.openlinksw.com/dataspace/owiki/wiki/UdaWikiWeb/ConfigureUDAFeatures][advanced configuration instructions]] and [[http://wikis.openlinksw.com/dataspace/owiki/wiki/UdaWikiWeb/ConfigureTroubleshootDBMSsForDataAccess][native database configuration]].

   * The UDA Wiki Web also provides links to [[http://wikis.openlinksw.com/dataspace/owiki/wiki/UdaWikiWeb/UDATroubleshootingResources][troubleshooting resources and interpretation of diagnostic resources]]. Basic error message information can also be found  in the general [[Technical FAQs]] and [[Mac FAQ]].
	
   * Questions pertaining to product capabilities may be addressed using our [[ProductComparisonSheet][UDA Driver Comparison Guide]].	

   * If you need additional assistance, feel free to open an [[http://support.openlinksw.com/support/online-support.vsp][online support case]] or visit our <A HREF="http://boards.openlinksw.com/support/">Discussion Forums</A>.


---++ Virtuoso Evaluators

   * The Virtuoso Universal Server and Client Connectivity Kit installers can be downloaded from the [[http://virtuoso.openlinksw.com/download/][Virtuoso Download Page]].

   * Our [[Installation Guides]] provide extensive installation instructions for the Universal Server and Client Connectivity Kit. Each Installation Guide also provides testing information and links to product pre-requisites docs.

   * The [[http://wikis.openlinksw.com/dataspace/owiki/wiki/VirtuosoWikiWeb/][Virtuoso Wiki Web]] covers a wide range of topics pertaining to Virtuoso configuration and usage.

   * The [[http://wikis.openlinksw.com/dataspace/owiki/wiki/OdsWikiWeb/][OpenLink Data Spaces (ODS)]] and [[http://wikis.openlinksw.com/dataspace/owiki/wiki/OATWikiWeb/][OpenLink AJAX Toolkit (OAT)]] Wiki Webs provide additional configuration and usage information for evaluators interested in the ODS application or OAT development kit. 

   * Our [[http://boards.openlinksw.com/support/][Discussion Forums]] provides multiple forums for evaluators with questions pertaining to Virtual & Native Data Management, Web Services, <nop>OpenLink Data Spaces, and the AJAX Toolkit (OAT).

   * If you need additional assistance, feel free to open an [[http://support.openlinksw.com/support/online-support.vsp][online support case]].
