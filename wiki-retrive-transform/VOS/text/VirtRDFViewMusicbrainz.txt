---++Musicbrainz Linked Data Views Deployment and Demo Scripts
Each Musicbrainz installation includes a demonstration database that includes a simple management system schema and instance data. This sample database provides an easy route for introducing Musicbrainz users and developers to the virtues of RDF based Linked Data Views built atop existing Musicbrainz relational databases.
