Choice of Labels Example

  1. Go to http://lod.openlinksw.com/sparql
  2. Enter the following query in the text area: 

    DEFINE  output:valmode   'LONG' 
DEFINE  input:inference  'facets' 
SELECT * 
  WHERE
    { 
      ?s  virtrdf:label  ?o 
    }
