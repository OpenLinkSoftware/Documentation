Virtuoso Open Source Edition Supported Platforms

The following comprises a list of platforms the Virtuoso Server has been tested and verified.

  • Linux Fedora Core 4 on Pentium
  • Linux Fedora Core 4 on AMD 64 (64 bit)
  • Mac OSX 10.4, 10.5, 10.6 Power PC & Intel 32 & 64 bit
  • Soalris x86 Sun Fire V20z Server, AMD Opteron CPU
  • Solaris Sparc 32 & 64 bit
