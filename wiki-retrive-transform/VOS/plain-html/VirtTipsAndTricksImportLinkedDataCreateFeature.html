How Can I Create Linked Data Resources via "Create" feature of ODS-Briefcase?

The guides that follow demonstrate how to create a Linked Data oriented Hypermedia Resource in the most basic form via ODS-Briefcase.

Prerequisites

The following VAD packages must be installed:

  • ODS-Framework
  • ODS-Briefcase

Create a WebDAV / Briefcase folder of type: Linked Data Import

Use instructions from the Linked Data Import setup guide to create folder from type: Linked Data Import. This folder type is bound to Virtuoso's back-end Graph Storage module (also known as the Quad Store).

Create Linked Data Resources using a variety of content formats from the RDF family

  • Create a Linked Data Resource with N-Triples content
  • Create a Linked Data Resource with Turtle content
  • Create a Linked Data Resource with RDF/XML content
  • Create a Linked Data Resource with RDFa content
