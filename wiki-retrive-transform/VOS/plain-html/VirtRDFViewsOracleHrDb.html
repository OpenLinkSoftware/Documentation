RDF based Linked Data Views atop Oracle's Human Resources Sample Database

Each Oracle installation includes a demonstration database that includes a Human Resources management system schema and instance data. This sample database provides an easy route for introducing Oracle users and developers to the virtues of RDF based Linked Data Views built atop existing Oracle relational databases.
