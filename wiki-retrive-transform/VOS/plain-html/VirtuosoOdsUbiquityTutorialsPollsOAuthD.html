Polls OAuth Ubiquity Authentication Example

The following example demonstrates oauth authentication getting the OAuth SID value for ex. for existing Polls instance "mypolls" with owner user demo at the OpenLink Demo server.

In order to execute correctly the example, you need to have OAuth Generate Key for the Polls "mypolls" instance at OpenLink Demo server. More information and simple scenario how to be done you can find here.

  1. Execute the command: 

    ods-oauth-host http://demo.openlinksw.com/OAuth

  2. Execute the command: 

    ods-set-mode oauth

  3. Go to http://demo.openlinksw.com/ods/oauth_sid.vsp
  4. In the shown form:
    1. Enter for user demo
    2. Choose the application "mypolls".
    3. Click the "Execute" button.
  5. Enter for user demo its password and click the "Login" button.
  6. Click the "Authorize" button.
  7. Copy the obtains OAuth SID value for ex.: c0655cf0c4b312a6a70188e4d8fba726
  8. Execute the following command: 

    ods-set-poll-oauth c0655cf0c4b312a6a70188e4d8fba726
