Ajax Database Connectivity (ADBC)

  • Architecture

Ajax Database Connectivity is a Generic Data Access API for JavaScript that provides transparent access to SQL (via XMLA), RDF (via SPARQL), XML (via HTTP), Web Services (SOAP or REST variants over HTTP), and Free Text (via GData). It provides query results in JSON or XML and includes built-in support for XSLT and XPath/XQuery should additional local post-processing be required.

XMLA is the SOAP or REST protocol used for SQL Data Access (as per the above). SPARQL, GData, Web Services (REST and SOAP) are the wire protocols for the respective query mechanisms.
