Latest Version

Release 2.7

Complete email announcement


Hi, 

OpenLink Software are pleased to announce release 2.7 of the 
OpenLink AJAX Toolkit (OAT). 

This release includes the following new features:

   * a new OAT.Win API / style 
   * a new Linked Data Resource save option for physical and virtual 
     RDF Graphs from DESCRIBE or CONSTRUCT based iSPARQL Queries
   * support for multiple protocol schemes: lsid, doi, oai, and urn 
     alongside http
   * more demonstrations

There are enhancements to a++ links and styles, the RDF-browser and 
iSPARQL applications, and the UI. 

There are fixes to URI parsing, N3 proxy support, loading path, parsing 
the month in the timeline control, the speed of drawing piecharts and 
other minor bug-fixes.

For further information, please see: 

Project homepage on sourceforge.net: 
   http://sourceforge.net/projects/oat 

File downloads: 
   http://sourceforge.net/projects/oat/files 

Live Features Demonstrations: 
   http://demo.openlinksw.com/DAV/JS/demo/index.html

Regards,

Succinct Freshmeat version


This release includes a new OAT.Win API and style, support in iSPARQL 
for loading .LDR resources, support for the OAI protocol and more 
demonstrations. There are enhancements to a++ links and styles, the 
RDF browser and iSPARQL applications and across the UI.  There are 
fixes to URI parsing, N3 proxy support, loading path, and faster 
drawing of piecharts amongst other bugfixes.

Release 2.6

Complete email announcement


Hi, 

OpenLink Software are pleased to announce release 2.6 of the 
OpenLink AJAX Toolkit (OAT). 

This release includes the following:

New Semantic Data Web Technologies

   * Javascript-based Fresnel processor enabling declarative RDF-based 
     display templates for RDF Data Sources. This the very first 
     implementation of a Core Fresnel Vocabulary & Extensions processor 
     for Javascript.
   * An XSLT template for producing HTML pages from the Fresnel 
     processor's XML output
   * Interactive Query Builder for SPARQL (iSPARQL). This version of 
     the iSPARQL application includes support for INSERTs and DELETEs.
   * Enhanced Javascript-based N3/Turtle parser
   * New Navigator UI for the OAT RDF Browser amongst other fixes and 
     enhancements

Other Items:

   * OAT.Notify widget and notification service, facilitating real-time 
     tracking of data sources. This is similar to services such as Yowl 
     (Javascript) and Growl (native Mac OS X)


For further information, please see: 

Project homepage on sourceforge.net: 
   http://sourceforge.net/projects/oat 

File downloads: 
   http://sourceforge.net/projects/oat/files 

Live Features Demonstrations:

Release 2.5

Complete email announcement


Hi, 

OpenLink Software are pleased to announce release 2.5 of the 
OpenLink AJAX Toolkit (OAT). 

This new version of OAT adds an SVG-based visual SPARQL Query builder  
("iSPARQL") to the rich collection of OAT-based applications. It also 
includes aesthetic enhancements to the OAT RDF Browser such as a

Hi, 

OpenLink Software are pleased to announce release 2.4 of the 
OpenLink AJAX Toolkit (OAT). 

This release introduces a greatly enhanced RDF Browser application

Release 1.0

Complete email announcement


Hi, 

OpenLink Software are pleased to announce the first release of the 
OpenLink AJAX Toolkit (OAT), version 1.0. OAT offers a broad 
Javascript-based, browser-independent widget set for building rich 
user-interfaces usable in all common browsers. It comes with sample 
applications for interactive SQL query composition and database design. 
These applications work against any Microsoft XML for Analysis (XMLA) 
servers, such as OpenLink Virtuoso and others. 

For further information, please see: 

Project homepage on sourceforge.net: 
   http://sourceforge.net/projects/oat
