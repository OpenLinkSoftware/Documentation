---+ OAT SPARQL Query By Example

---++ iSPARQL Dymanic Data Web Page file type (.isparql)

Saved files of this type (.isparql) can be called with additional parameters:
   * <code>showQuery</code> -- this will show an extra tab that shows the executed query
   * <code>showRequest</code> -- this will show the ajax request send to the endpoint
   * <code>hideResponse</code> -- this will show the raw ajax response from the endpoint
